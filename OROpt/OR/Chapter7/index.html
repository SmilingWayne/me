    з®ҖеҚ•зҗҶи§ЈпјҢжҲ‘д»¬жҠҠеҺҹй—®йўҳзҡ„еҸіз«ҜйЎ№з§»еҲ°зӣ®ж ҮеҮҪж•°дёҠпјҢзӣ®ж ҮеҮҪж•°зҡ„еҶізӯ–еҗ‘йҮҸй•ҝеәҰе°ұжҳҜ\(d\)зҡ„й•ҝеәҰпјҢд№ҹе°ұжҳҜиҠӮзӮ№зҡ„ж•°йҮҸгҖӮиҖҢеҸіз«ҜйЎ№еӨ§йғЁеҲҶжҳҜ0пјҢжҲ‘д»¬з”Ё \(l_v\) иЎЁзӨәеҺҹй—®йўҳжҜҸдёӘзәҰжқҹпјҲе…¶е®һе°ұжҳҜжҜҸдёӘиҠӮзӮ№ \(v\) пјүеҜ№еә”зҡ„еҜ№еҒ¶еҶізӯ–еҸҳйҮҸгҖӮеҸҜд»ҘзңӢеҲ°йҷӨдәҶstart / terminal еӨ–пјҢе…¶д»–иҠӮзӮ№зҡ„ \(l_v\) йғҪжҳҜ0пјҢжүҖд»ҘеҜ№еҒ¶й—®йўҳзӣ®ж ҮеҮҪж•°еҚідёәпјҡ

    \[\max l_s - l_t\]

    е…¶дёӯ \(l_t\) еҜ№еә” terminal иҠӮзӮ№зҡ„еҜ№еҒ¶еҸҳйҮҸпјҢ\(l_s\) еҜ№еә” start иҠӮзӮ№зҡ„еҜ№еҒ¶еҸҳйҮҸгҖӮ

    зҺ°еңЁејҖе§ӢеҶҷеҜ№еҒ¶й—®йўҳзҡ„зәҰжқҹжқЎд»¶гҖӮ

    з”ұдәҺеҺҹй—®йўҳзҡ„зі»ж•°зҹ©йҳөжң¬иә«жҳҜиҫ№зҡ„е…іиҒ”зҹ©йҳөпјҢиҖҢеҺҹй—®йўҳзҡ„жҜҸдёҖеҲ—д»ЈиЎЁдёҖжқЎиҫ№еҮәе…ҘиҠӮзӮ№зҡ„жғ…еҶөпјҢеҸӘжңүдёҖдёӘ\(-1\)е’Ң\(1\)пјҢе…¶дҪҷеқҮдёә\(0\)пјҢиҝҷдёӘз»“жһ„иҪ¬зҪ®еҪўжҲҗеҜ№еҒ¶й—®йўҳзҡ„зі»ж•°зҹ©йҳөеҗҺпјҢеҜ№еә”еҜ№еҒ¶й—®йўҳеҶізӯ–еҸҳйҮҸзҡ„зі»ж•°пјҢжүҖд»ҘеҜ№еҒ¶й—®йўҳзҡ„зәҰжқҹдёӯпјҢжҜҸдёҖиЎҢеҸӘжңүдёҖдёӘ\(-1\)е’Ң\(1\)пјҢе…¶дҪҷзі»ж•°дёә\(0\). иҪ¬зҪ®еҗҺзҡ„иЎҢж•°еҜ№еә”еҺҹй—®йўҳзҡ„еҲ—пјҢд№ҹе°ұиҜҙжҳҺпјҢеҜ№еҒ¶й—®йўҳзҡ„зәҰжқҹдёӘж•° = иҫ№зҡ„дёӘж•°гҖӮ

    жүҖд»ҘпјҢжңҖзҹӯи·Ҝй—®йўҳзҡ„еҜ№еҒ¶й—®йўҳжңҖз»ҲеҸҜд»ҘеҶҷжҲҗпјҡ

    \[\max l_s - l_t\]
    \[\begin{aligned}
\begin{cases}
\begin{align}
l_p - l_q \leq c_{(p,q)}, \quad \forall (p,q) \in E \quad \\
l_v \geq 0, \quad \forall v \in V \quad 
\end{align}
\end{cases}
\end{aligned}\]

    \(l_p\) е’Ң \(l_q\) жҳҜз”ұжҜҸдёӘиҫ№ \((p,q)\) зҡ„иҠӮзӮ№еҶіе®ҡзҡ„пјҢеҜ№еә”зҡ„жҳҜеҺҹй—®йўҳзҡ„зәҰжқҹжқЎд»¶йҮҢдёҺиҠӮзӮ№ \(p\) е’Ң \(q\) жңүе…ізҡ„еҜ№еҒ¶еҸҳйҮҸгҖӮ

    дёҖдёӘеҗҜеҸ‘ејҸзҡ„зҗҶи§ЈжҖқи·ҜгҖӮ

    еҸҜд»ҘжҠҠеҺҹжқҘзҡ„еӣҫдёӯзҡ„жҜҸдёҖдёӘиҫ№жғіиұЎжҲҗдёҖдёӘж— жі•иў«жӢүдјёзҡ„зәҝпјҢд»ҘиҠӮзӮ№зӣёиҝһжҺҘпјҢиҫ№ \(e\) зҡ„й•ҝеәҰжҳҜ \(c_e\)пјҢзҺ°еңЁдёҖеҸӘжүӢжӢҺдҪҸиө·зӮ№ \(s\)пјҢдёҖеҸӘжүӢжӢҺдҪҸз»ҲзӮ№ \(t\) пјҢе°ҪеҸҜиғҪең°жҠҠиҝҷдёӘжҸҗзәҝжңЁеҒ¶дёҖж ·зҡ„дёңиҘҝжҚӢзӣҙгҖҒжҚӢйЎәпјҢпјҲд№ҹеҸҜд»ҘзҗҶи§ЈдёәжӢҺзқҖиө·зӮ№пјҢи®©еӣҫиҮӘз„¶дёӢеһӮпјүгҖӮ

    жӯӨж—¶жңүдёҖдәӣиҫ№еӣ дёәжҜ”иҫғй•ҝпјҢдјҡжӮ¬жҢӮеңЁиҫ№дёҠпјҲжҜ”еҰӮдёӢеӣҫдёӯзҡ„иҫ№ \((1,3), (3,5), (5,4)\)пјҢжңүдёҖдәӣиҫ№иҮӘз„¶дёӢеһӮ \((1,2), (2,5)\)гҖӮ

    зӯүиҝҷеј еӣҫдёҚиғҪеҶҚз»§з»ӯдёӢеһӮпјҲдёҚиғҪз»§з»ӯиў«дҪ жӢүй•ҝжҚӢеҫ—жӣҙйЎәдәҶпјүпјҢжӯӨж—¶д»Һ \(s\) еҲ° \(t\) зҡ„и·қзҰ»дёҖе®ҡжҳҜвҖңжүҖжңүзәҝеӨ„дәҺз»·зӣҙзҡ„и·Ҝеҫ„вҖқдёӯжңҖй•ҝзҡ„зҡ„йӮЈдёӘи·Ҝеҫ„пјҲжңҙе®һең°еҪўе®№е°ұжҳҜпјҡжҲ‘д»¬ж— жі•еҶҚжҚӢд»»дҪ•дёҖжқЎиҫ№дҪҝеҫ—д»ҺsеҲ°tиҮӘз„¶дёӢеһӮж—¶жӣҙй•ҝдәҶпјүпјҢиҝҷе°ұеҜ№еә”зҡ„жҳҜжҲ‘д»¬зҡ„еҜ№еҒ¶й—®йўҳпјӣ

    еҸҚиҝҮжқҘжқҘжғіпјҢиҝҷжқЎд»Һ \(s\) еҲ° \(t\) зҡ„и·Ҝеҫ„пјҢд№ҹе°ұжҳҜд»Һ \(s\) еҲ° \(t\) зҡ„жңҖзҹӯи·ҜпјҢеӣ дёәеҰӮжһңдёҚиө°иҝҷдёӘи·ҜпјҢеҝ…йЎ»иҰҒд»ҺйӮЈдәӣжӮ¬жҢӮзқҖзҡ„иҫ№дёҠиө°пјҢиҝҷдәӣиҫ№жҳҺжҳҫжІЎжңүиў«жӢүзҙ§пјҢд№ҹе°ұдёҖе®ҡдёҚдјҡжҜ”жҲ‘иҝҷжқЎи·ҜжӣҙзҹӯдәҶгҖӮиҝҷз§Қж–№ејҸдёӢпјҢеҜ№еә”зҡ„е°ұжҳҜжҲ‘д»¬зҡ„еҺҹй—®йўҳпјҡжңҖзҹӯи·Ҝй—®йўҳгҖӮ

    еҸӮиҖғй“ҫжҺҘпјҡMIT | еҸӮиҖғй“ҫжҺҘпјҡStanford

    жұӮи§ЈжңҖзҹӯи·Ҝй—®йўҳпјҡ Dijkstra з®—жі•пјҲеҚ•жәҗжңҖзҹӯи·Ҝпјүвҡ“пёҺ

    е…ідәҺDijkstraз®—жі•

    дёҖз§Қз”ЁдәҺи®Ўз®—д»ҺеҚ•дёӘжәҗиҠӮзӮ№еҲ°еӣҫдёӯжүҖжңүе…¶д»–иҠӮзӮ№зҡ„жңҖзҹӯи·Ҝеҫ„зҡ„з®—жі•гҖӮйҖӮз”ЁдәҺеҠ жқғеӣҫпјҢе…¶дёӯиҰҒжұӮиҫ№зҡ„жқғйҮҚдёәйқһиҙҹеҖјгҖӮ

    • Dijkstraз®—жі•йҮҮз”Ёзҡ„жҳҜдёҖз§Қ==иҙӘеҝғ==зҡ„зӯ–з•ҘгҖӮеЈ°жҳҺдёҖдёӘеҲ—иЎЁdisжқҘдҝқеӯҳжәҗзӮ№еҲ°еҗ„дёӘйЎ¶зӮ№зҡ„жңҖзҹӯи·қзҰ»пјҢе№¶з”ЁдёҖдёӘйӣҶеҗҲ \(T\) дҝқеӯҳе·Із»ҸжүҫеҲ°дәҶжңҖзҹӯи·Ҝеҫ„зҡ„йЎ¶зӮ№зҡ„йӣҶеҗҲгҖӮжҜҸж¬ЎйғҪд»ҺеҪ“еүҚиғҪжҠөиҫҫзҡ„иҠӮзӮ№дёӯзҡ„жңҖиҝ‘иҠӮзӮ№еҮәеҸ‘пјҢз»§з»ӯдёӢдёҖжӯҘеҜ»и·Ҝзҡ„иҝҮзЁӢпјҢзӣҙеҲ°жҠөиҫҫзӣ®зҡ„ең°гҖӮ
    жӯҘйӘӨ
    1. еҲқе§Ӣж—¶пјҢжәҗзӮ№ \(s\) зҡ„и·Ҝеҫ„жқғйҮҚиў«иөӢдёә 0 гҖӮиӢҘеҜ№дәҺйЎ¶зӮ№ \(s\), еӯҳеңЁиғҪиө°иЎҢзҡ„иҫ№ \((s,m)\) пјҲж„ҸжҖқжҳҜеҸҜд»ҘеҲ°иҫҫиҠӮзӮ№ \(m\)пјүпјҢеҲҷжҠҠ dis[m]и®ҫдёә \(c_{sm}\)пјҢеҗҢж—¶жҠҠеҲ°жүҖжңүе…¶д»–зҡ„иҠӮзӮ№пјҲд№ҹе°ұжҳҜ\(s\)еңЁиҝҷдёҖжӯҘж— жі•еҲ°иҫҫзҡ„иҠӮзӮ№пјүзҡ„й•ҝеәҰи®ҫдёәж— з©·еӨ§гҖӮиҝҷдёҖжӯҘйӘӨпјҢйӣҶеҗҲTдёӯд»…еҢ…еҗ«жәҗзӮ№гҖӮ
    2. д»ҺеҲ—иЎЁdisдёӯйҖүжӢ©жңҖе°ҸеҖјпјҢд»ҘеҸҠиҝҷдёӘжңҖе°ҸеҖјеҜ№еә”зҡ„иҠӮзӮ№пјҲи®°дёә \(z\)пјүгҖӮиҜҘеҖје°ұжҳҜжәҗзӮ№ \(s\) еҲ°иҜҘиҠӮзӮ№ \(z\) зҡ„жңҖзҹӯи·Ҝеҫ„гҖӮжҠҠиҜҘзӮ№еҠ е…ҘеҲ° \(T\) дёӯгҖӮOKпјҢжӯӨж—¶е·Із»ҸжүҫеҲ°дәҶдёҖдёӘзӮ№зҡ„жңҖзҹӯи·ҜпјҢдҪҶиҝҳжІЎеҲ°иҫҫзӣ®ж ҮиҠӮзӮ№гҖӮ
    3. иҝҷж—¶пјҢжҲ‘д»¬йңҖиҰҒзңӢпјҢж–°еҠ е…Ҙзҡ„йЎ¶зӮ№ \(z\) жҳҜеҗҰеҸҜд»ҘеҲ°иҫҫе…¶д»–иҠӮзӮ№пјҢе№¶дё”зңӢзңӢйҖҡиҝҮиҜҘиҠӮзӮ№еҲ°иҫҫе…¶д»–зӮ№зҡ„и·Ҝеҫ„й•ҝеәҰжҳҜеҗҰжҜ”еҪ“еүҚdisдёӯеӯҳеӮЁзҡ„вҖңеҪ“еүҚжңҖзҹӯи·ҜвҖқиҰҒжӣҙзҹӯгҖӮеҰӮжһңжҳҜпјҢйӮЈд№Ҳе°ұжӣҙж–°иҝҷдәӣйЎ¶зӮ№еңЁ dis дёӯзҡ„еҖјгҖӮ
    4. жӣҙж–°е®ҢжҜ•еҗҺпјҢз»§з»ӯйҮҚеӨҚ 3, зӣҙеҲ° \(T\) дёӯеҢ…еҗ«дәҶеӣҫзҡ„жүҖжңүйЎ¶зӮ№гҖӮ

    жіЁж„ҸпјҒ

    • ж— жі•еӨ„зҗҶиҙҹжқғй—®йўҳгҖӮ
    • иҠӮзӮ№иҫғеӨҡзҡ„иҜқпјҢи®Ўз®—йҮҸдјҡеҠ еӨ§гҖӮ

    дјӘд»Јз ҒеҰӮдёӢпјҡ

    еҪ“з„¶пјҢеңЁзј–зЁӢе®һзҺ°зҡ„иҝҮзЁӢдёӯпјҢдјҡж¶үеҸҠеҲ°пјҡд»ҺеҪ“еүҚжүҖжңүеҸҜиҫҫиҠӮзӮ№дёӯйҖүжӢ©и·Ҝеҫ„жңҖзҹӯзҡ„йӮЈдёӘиҠӮзӮ№зҡ„иҝҮзЁӢпјҢдёҖиҲ¬иҖҢиЁҖдјҡйҖҡиҝҮдјҳе…ҲйҳҹеҲ—жқҘе®һзҺ°гҖӮ

    import heapq
def dijkstra(graph, start, end):
    """
    е®һзҺ°Dijkstraз®—жі•пјҢи®Ўз®—д»Һиө·зӮ№еҲ°з»ҷе®ҡз»ҲзӮ№зҡ„жңҖзҹӯи·Ҝеҫ„е’Ңи·Ҝеҫ„й•ҝеәҰгҖӮ
    """
    # еҲқе§ӢеҢ–и·қзҰ»еӯ—е…ёе’ҢеүҚй©ұиҠӮзӮ№еӯ—е…ё
    distances = {vertex: float('infinity') for vertex in graph}
    distances[start] = 0
    predecessors = {vertex: None for vertex in graph}
    # дҪҝз”Ёдјҳе…ҲйҳҹеҲ—жқҘеӯҳеӮЁе’ҢиҺ·еҸ–еҪ“еүҚе·ІзҹҘзҡ„жңҖзҹӯи·қзҰ»зҡ„иҠӮзӮ№
    priority_queue = [(0, start)]
    while priority_queue:
        current_distance, current_vertex = heapq.heappop(priority_queue)
        # еҰӮжһңеҲ°иҫҫз»ҲзӮ№пјҢжһ„е»әи·Ҝеҫ„е№¶иҝ”еӣһз»“жһң
        if current_vertex == end:
            path = []
            while current_vertex is not None:
                path.append(current_vertex)
                current_vertex = predecessors[current_vertex]
            path.reverse()  # йҖҶеәҸ
            return current_distance, path
        # еҰӮжһңд»Һдјҳе…ҲйҳҹеҲ—дёӯеҸ–еҮәзҡ„и·қзҰ»еӨ§дәҺеҪ“еүҚи®°еҪ•зҡ„и·қзҰ»пјҢз»§з»ӯ
        if current_distance > distances[current_vertex]:
            continue
        # йҒҚеҺҶеҪ“еүҚиҠӮзӮ№зҡ„жүҖжңүйӮ»жҺҘиҠӮзӮ№
        for neighbor, weight in graph[current_vertex].items():
            distance = current_distance + weight
            # еҸӘжңүеҪ“и®Ўз®—еҮәзҡ„ж–°и·қзҰ»е°ҸдәҺеҪ“еүҚи®°еҪ•зҡ„и·қзҰ»ж—¶пјҢжүҚжӣҙж–°и·қзҰ»е’ҢеүҚй©ұиҠӮзӮ№пјҢе№¶е°ҶйӮ»жҺҘиҠӮзӮ№ж·»еҠ еҲ°дјҳе…ҲйҳҹеҲ—
            if distance < distances[neighbor]:
                distances[neighbor] = distance
                predecessors[neighbor] = current_vertex
                heapq.heappush(priority_queue, (distance, neighbor))
    return float('infinity'), []  # еҰӮжһңз»ҲзӮ№дёҚеҸҜиҫҫпјҢиҝ”еӣһж— з©·еӨ§и·қзҰ»е’Ңз©әи·Ҝеҫ„

# жөӢиҜ•з”Ёзҡ„еӣҫж•°жҚ®
graph = {
    'A': {'B': 1, 'C': 4},
    'B': {'A': 1, 'C': 2, 'D': 5},
    'C': {'A': 4, 'B': 2, 'D': 1, 'E': 3},
    'D': {'B': 5, 'C': 1, 'E': 5},
    'E': {}
}

# и°ғз”ЁDijkstraз®—жі•
start_node = 'A'
end_node = 'E'
distance, path = dijkstra(graph, start_node, end_node)

# иҫ“еҮәз»“жһң
print(f"д»Һиө·зӮ№ {start_node} еҲ°з»ҲзӮ№ {end_node} зҡ„жңҖзҹӯи·Ҝеҫ„и·қзҰ»дёәпјҡ{distance}")
