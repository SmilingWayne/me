## ◆ 爱默生

!!! note ""
    爱默生经常和他的朋友梭罗、霍桑等人举行小型聚会，探讨神学、哲学和社会学问题。这种聚会当时被称为“超验主义俱乐部”，爱默生也自然而然地成为超验主义的领袖。他于1836年出版处女作《论自然》，在文学上的贡献主要在散文和诗歌方面。
