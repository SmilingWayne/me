# 冯至的诗

!!! quote ""
    <span style="color:red;font-weight:bold">在漫长的岁月里忽然有：彗星的出现、狂风乍起</span>

    <span style="color:red;font-weight:bold">我们整个的生命在承受：狂风乍起、彗星的出现。</span>

    —— 《我们准备着》


我觉得他们好像从古来 / 就一任眼泪不住地流 / 为了一个绝望的宇宙。 —— 《原野的哭声》

千年的梦像一个老人 / 期待着最好的儿孙 —— / 如今有人飞向星辰 / 却忘不了人世的纷纭。 —— 《一个旧日的梦想》

我们只在黎明和黄昏 / 认识了你是长庚、是启明 / 到夜半你和一般的星星 / 也没有区分 —— 《蔡元培》

万物都在享用你的那句名言 / 它道破一切生的意义 / 死和变 —— 《歌德》

几个贫穷的人低着头 / 在贫穷的房里剥土豆 / 却像是永不消融的冰块 —— 《梵高》

一生里有几回春几回冬 / 我们只感受时序的轮替 / 感受不到人间规定的年龄 —— 《别离》

暴雨把一切又淋入泥土，只剩下这点微弱的灯红，在证实我们生命的暂住。 —— 《我们听着狂风里的暴雨》

不要觉得一切都已熟悉 / 到死抚摸着自己的发肤 / 生了疑问：这是谁的身体？ —— 《我们天天走着一条小路》
