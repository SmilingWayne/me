# 特洛伊与三大史诗

这个部分想做很久了：为的就是看书时候不再痛苦，于是有了下面这种分专题的笔记方式。感谢 Claude。

## 特洛伊的木马：「伊利亚特」

特洛伊之战 （Trojan War），一言以蔽之，两边人打仗：希腊联军  vs 特洛伊守军。久攻不下，希腊人佯装撤退，留下一个巨大木马，特洛伊人觉得希腊人战败了就把木马当作战利品拖进城里，没想到希腊人在木马里藏了一群战士，趁夜晚特洛伊人睡着后疲累之时冲出来攻破了特洛伊城。当然《伊利亚特》并没有完整地记录和讲述上面这个故事，一些后续未见于其中，为了方便，这里全部放到这篇笔记中。

!!! note "一个私货"
    史诗故事之所以看起来又臭又长，有一个十分致命的原因是**人物称呼的多变性**。作者有时候会把当事人的头衔依次排开，有时候，一方又会因为和另一方的特定关系（母子、出生地、阵营、朋友、恩仇）而采取不同的方式进行指代。不少出场人物众多的作品通常会通过划分更多阵营、国家形成人物小团体以避免太多的称呼交叉，但是希腊史诗尤其是特洛伊之战中，阵营对立极度二元化，这就构成了史诗中的某种阅读灾难。比如《伊利亚特》常以**父名+本名+族群/装束/地域修饰语**的组合方式称呼人物，举个例子：

    > "佩琉斯的英勇的儿子、忒提斯善战的孩子，穿着精美胫甲的阿开奥斯人的战士"

    这仨实际上指的是同一个人**阿喀琉斯** （~~主席台坐不下这么多人！~~），诶又有时候他又不会列这么长：

    比如 Z 城邦有个母亲 A 与父亲 B 有儿子 C，儿子 C 认识挚友 D，此时父亲 B 或许就称呼孩子为：“啊！Z 那冲动的孩子，A 那不服管教的儿子啊！”，而母亲 A 又会称呼儿子为 “啊！B 那可敬的孩子！”，但是挚友 D 可能直呼为 “来自 Z 的那勇猛的战士啊！”

    于是你的大脑就乱糟糟的不知所云。因此熟悉（至少是有所记忆地去判断）人物的父母、配偶、子女关系其实还有点小用的。

### 阵营：特洛伊

<span style="color:red;font-weight:bold">特洛伊</span>（Troy），又被称为伊利昂（Ilium），荷马史诗《伊利亚特》（*Iliad*）书名由此而来。特洛伊的**位置**：小亚细亚西北部（今土耳其境内），**国王：普里阿摩斯**（Priam），**王后：赫卡柏**（Hecuba）。**结局**：城破国亡，王室成员死的死、沦为奴隶的沦为奴隶。

因为特洛伊是守城的一方（没啥联军的一方），也是战败的一方，所以这边的人物结局相对还挺好记的，因为很大比例战后 / 战中就被干掉了，提前下线省去了很多故事，一个例外是关键人物**埃涅阿斯**，后续关于他还有一长传津津乐道的风流故事。

| 人物                                                     | 英文      | 身份                       | 关键信息                                                                                                                                                             |
| -------------------------------------------------------- | --------- | -------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **普里阿摩斯**                                           | Priam     | 特洛伊国王                 | 城破后被杀                                                                                                                                                           |
| **赫卡柏**                                               | Hecuba    | 特洛伊王后                 | 战后沦为奴隶，女儿波吕克塞娜被献祭于阿喀琉斯墓前，幼子波吕多洛斯被邻国国王波吕墨斯托耳（Polymestor）见财忘义杀害。赫卡柏向阿伽门农求得复仇权，亲手杀死波吕墨斯托耳。 |
| **赫克托耳**                                             | Hector    | 特洛伊统帅、王子           | 特洛伊第一勇士，杀死帕特洛克罗斯，后被阿喀琉斯所杀                                                                                                                   |
| <span style="color:red;font-weight:bold">埃涅阿斯</span> | Aeneas    | 特洛伊王族、英雄           | 战中丧妻，最终逃离特洛伊，高光在其逃出后的故事，后来维吉尔史诗《埃涅阿斯纪》（Aeneid）的主角。                                                                       |

| **波吕多洛斯**                                           | Polydorus | 特洛伊幼子                 | 战时被送往邻国避难，城破后被邻国国王波吕墨斯托耳杀害夺财                                                                                                             |
| **潘达罗斯**                                             | Pandarus  | 特洛伊将领                 | 特洛伊第一神箭手，远射神阿波罗（Apollo）曾将神弓赠予他。射伤墨涅拉奥斯（Menelaus），导致停战协议破裂，后在战场上被迪奥墨德斯（Diomedes）刺死。                       |
| **欧福尔波斯**                                           | Euphorbus | 特洛伊将领                 | 在战场上被墨涅拉奥斯（Menelaus）杀死，墨涅拉奥斯剥取其铠甲作为战利品。                                                                                               |

### 阵营：希腊城邦联军

到了城邦联军这里<u>情况就变得十分混乱</u>，因为各种名字层出不穷并且初读会有极大的痛感，==首先彼时城邦之间大伙不叫“希腊”，而是自称"阿开奥斯人（Achaeans）"==，我甚至觉得这算是这个故事最重要的注释。

然后就是主要的城邦，最重要的几个已经高亮了，因为后续有无数的故事围绕这几个人展开：

| 城邦                      | 君主/代表人物             | 备注                     |
| ------------------------- | ------------------------- | ------------------------ |
| ==**迈锡尼（Mycenae）**== | ==阿伽门农==（Agamemnon） | 希腊联军统帅             |
| ==**斯巴达（Sparta）**==  | 墨涅拉奥斯（Menelaus）    | 阿伽门农之弟，海伦之夫   |
| **阿尔戈斯（Argos）**     | 迪奥墨德斯（Diomedes）    | 勇猛善战                 |
| **伊塔卡（Ithaca）**      | ==奥德修斯==（Odysseus）  | 足智多谋，木马计的策划者 |
| **弗提亚（Phthia）**      | ==阿喀琉斯==（Achilles）  | 希腊第一勇士，弱点在脚踝 |

以上是一些有头有脸的人物，联军阵营这边的人头非常多而且比较难记，身世罗列如下：
