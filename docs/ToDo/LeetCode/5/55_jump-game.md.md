=== "Python"

    ```Python
    class Solution:
        def canJump(self, nums: List[int]) -> bool:
            if len(nums) == 1:
                return True
            res = nums[0]
            for i in range(1, len(nums)):
                if i > res:
                    return False
                if i <= res:
                    res = max(res, i + nums[i])
            return res >= len(nums) - 1
    ```
