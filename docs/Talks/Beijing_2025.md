# 帝京 Citywalk / Solotrip 小记 (2025年版)

!!! note "说在前面"
    笑笑的用户画像：文艺拥趸、博物馆爱好者、citywalk 发烧友、公共交通探险家，迷信导航软件的路痴（~~尽管研究方向是交通工程~~）、小众派。记录一下2025年在北京 solo trip 过的那些个小地方：有冷门有热门。

- [x] **地坛公园**：在北京住下后去的第一个公园，宁静致远：每个人都该写写自己的《我与地坛》；
- [x] **雍和宫**，可以准备工作做细一点，第一次逛没做功课导致错过了很多东西（🤷），主打一个祈福也要讲究基本法，不要像我一样乱拜一气，最后也不知凑效了没有：贪多勿得；
- [x] **孔庙和国子监**：就记得超级多石碑...真的好多石碑...
- [x] **中国美术馆**：国字号，老建筑了，还不错，记得先去6楼看藏宝阁；
- [x] **三联韬奋书店**：在北京第一个种草书店，就在美术馆边上，很不错的去处，记得第一次去就坐到晚上了 ... 小小槽点是没啥座位，捡个垫子席地而坐，看书久了伤腰；
- [x] **玉渊潭公园**：和友人一同去的，能赏樱花，能看候鸟，这个名字很喜欢；
- [x] **中国人民大学**：找友人玩耍时去的，去的那个春日杨絮瞎飞，但是景致很好，尤其记得明德楼，雄伟的红墙、喝了咖啡，闯进图书馆溜达；
- [x] **怀柔区水长城**：团建去的，一个既能够划船又能够爬长城的神奇的地方（👀），就记得坐了缆车，就记得有一个什么树的保护林；
- [x] **奥森公园**：我戏称“氧气公园” ... 个人是骑车+步行+休息结合，多人适合野营有木有！很适合嗅闻生命力吸氧，去班味儿必备，沿途能逛老多地方了：南下就是鸟巢国家体育场、水立方、奥体中心，奥运味儿蹭一下就来了您瞅瞅；
- [x] **北海公园**：看了一个西藏塘嘎的艺术展：矿物质颜料和精细的刻磨；第一次去没有看白塔，光顾着看老爷爷画风景画了，没想到后来也没找到机会再拜之；
- [x] **日坛公园**：忘了有啥特色了，就记得怎么这么多外国人一看地图原来边上是好多外国大使馆，返程时候特意到朝鲜大使馆门口溜达了一圈看兵哥哥巡逻（也不知道图啥）；
- [x] **芳草地**：一个很艺术的商业街，风格前卫，最上层有个艺术展厅（就记得很前卫），但是没啥人，世贸天阶。
- [x] **什刹海**：买了梵高的海报可以贴在墙上，很松弛，适合看落日，还有钓鱼游泳的老爷爷，同样很适合打卡爱好者；
- [x] **中国工艺美术馆**：高于预期的一个馆，本来没抱多少期待，逛了感觉值得一去，精致的陶瓷陶塑和装饰艺术、紫砂、玉雕（链条：在一块小玉上扭转雕刻而成）、汉白玉、金属镶嵌画、漆画、雕刻工艺（牙雕：象牙上的雕刻）
- [x] **清北/五道口/六道口大学城**：某次15号线坐到底后的 city ride！还有不少很多中科院研究所。
- [x] **中国考古博物馆**：引导一开始有些模糊，但是展品分量足够大，甲骨文、青铜器、陶器都有，殷墟妇好墓，二里头；
- [x] **国家典籍博物馆**：很多古代典籍著作的真品：宋代和清代的刻本，甲骨文。就在国家图书馆旁边；
- [x] **中国国家图书馆**：馆分南北，气势雄伟，很多藏书，办了卡（带身份证即可，免费）就能进去自习，借书要给押金；
- [x] **龙潭公园**：这里的动物似乎都不怕人；鸟儿从步道上横飞而过，白鹅上岸了，靠在人的吊床边，野鸭也是自由的，束缚他们的或许只有人造的堤岸。他们不知道，那一阵阵用指掌划开的波浪涟漪，竟然让我心里荡起了微澜：野鸭虽晚不须归。
- [x] **中国钱币博物馆**：好多钱钱钱钱币币币币钞钞钞钞、由于靠近天安门和人民大会堂，沿路安保很严格，路不是很通顺，在烈日下的胡同里徒步了1km并被保安叔叔拿着身份证对比了很久；
- [x] **风入松书店**：氛围意外地很好，有咖啡需消费，大多数书都有拆封的可以直接看，而且看起来似乎是个老店翻新了，比较推荐；
- [x] **前门的pageone书店**：三楼消费落座，有些小热，空调不是那么给力，有一个小熊汉斯的玩偶很可爱，在门口徘徊一分钟后买之挂包；
- [x] **妙应寺白塔**：一个元代由尼泊尔工匠设计建造的佛塔，在阜成门；还看了一个唐卡艺术展（藏传佛教？天王、持国、布袋和尚、广目天王、救难八度母、伞盖佛母、金刚，八十四大成就者）
- [x] **天主教西什库座堂**：没有能进去看，感觉一般般，也可能是没拍到好照片的缘故：讲一些天主教的故事！
- [x] **红楼藏书馆**：很适合看书，而且很多老书，去了两三次...
- [x] **北京图书大厦**：诶，很不错的地方，西单地铁站一出来就是。
- [x] **纸上声音书店**：小资到顶峰了，我一直呆到晚上；
- [x] **798艺术街区**：非常前卫；
- [x] **景山公园**：中轴线公园，门票两块钱（笑），值得一看，感觉外国人好像很爱来这儿；
- [x] **王府井大街**：跟小学读到的王府井感觉一毛一样！约友人一同逛更好玩；
- [x] **北京艺术博物馆**：实际上是万寿寺附近景观博物馆，有一些佛教的文物，和一些祝寿吉祥的物件，2020年才重新考古挖掘（耄耋：猫蝶；福：蝙蝠）无意中还看到了家具展、鸟禽展，比较新；
- [x] **中国海关博物馆**：很多珍贵的走私物品，虎皮熊皮走私，还有历史介绍、有一艘船902缉私艇，在建国门边上、旁边是古观象台（很难想象二环内竟然有一条船，而且是小平爷爷南巡讲话坐过的船船）；
- [x] **法国文化中心**：有一些法语书，有小孩；
- [x] **团结湖公园**：另一个“野鸭虽晚不须归”的地方，呼家楼儿。
- [x] **紫竹院公园**：很多竹，就在国家图书馆旁边，8月下旬去感觉很凉快；
- [x] **朝阳公园**：一个人去的，感觉有点子无聊，不知道是不是因为里面都是儿童设施的缘故；
- [x] **前门大街**：逛就完事了，似乎是个微雨日去逛的，逛一半坐下来把论文的revised script给写了..(sory...)
- [x] **望京 SOHO**：我逢人必讲那个关于扎哈的梗 ... 
- [x] **西二旗**：赋班味之地；
- [ ] **潘家园旧货市场/古玩市场**：挺有意思的市场，遗憾的是去的那天是最后顺路捎过去看得，时间有点晚，而且逛得浮光掠影（耸肩）；
- [ ] **国贸CBD**：路过了 N 次都没认真观察过的一个 CBD：“怎么着也得从‘金台夕照’站出来一次吧！”
- [ ] **南锣鼓巷**：因为没约到几个友人一同逛而总是有点遗憾的地方；
- [ ] **国家自然博物馆**：一直没去逛；
- [ ] **古建筑博物馆**（没有去因为最值得一看的藻井在维护！！）
- [ ] 红砖美术馆：就记得门票有点子贵；
- [ ] **中国电影博物馆**：边上有圈铁道，每次都说798和这儿可以一起逛，结果就逛了798；

有一些热得发紫的地方以前来北京就去过了，所以不列在上面，还有些是不记得有没有去过的：

- [ ] **故宫**：大大大大大！我说不清北京的气质像故宫还是故宫的气质像北京，反正就大大大大大。
- [ ] **八达岭长城**：第一次爬是冒着雨哼哧哼哧爬的；
- [ ] **颐和园、圆明园**：清北边上，很适合散心，top-2就是吃得好哇！
- [ ] **国博**：一定要找机会去逛的馆子；
- [ ] **军事博物馆**：要预约，每次班味浓的时候都忘记约，大飞机大炮大枪；
- [ ] **天坛公园**：人多多，想出片是要点技术的：国泰民安的美！
- [ ] **环球影城**：哈利波特的禁忌之旅！
