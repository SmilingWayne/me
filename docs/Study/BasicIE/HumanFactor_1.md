# 人因工程

## Part 1: 人因工程学概述

### 人因工程学的定义
人因工程学（Human Factors Engineering）是将心理学、生理学、解剖学、管理学以及各种工程学知识与方法相结合，探讨人、工作和生活彼此关系的新领域，主要功能在于研究如何设计一个最适合人的生活及工作的新环境。

**国际人机工程学会定义**：
- 研究人在某种工作环境中的解剖学、生理学和心理学等方面的因素
- 研究人和机器及环境的相互作用
- 研究在工作中、生活中和休息时怎样统一考虑工作效率、人的健康、安全和舒适等问题

### 人因工程学的特点
- **用户中心**：产品设计始终把用户放在首位，并考虑生产者和相关者
- **个体差异**：意识到个体在能力和限制上的差异，并充分考虑这些差异对设计的影响
- **数据驱动**：强调设计过程中经验数据和评价的重要性，依靠科学方法使用客观数据
- **系统观点**：用系统观点考虑问题，意识到事物、过程、环境和人都不是独立存在的

### 人因工程学的历史发展
1. **第一阶段：19世纪晚期和20世纪初**  
   - 工业革命与科学管理时期  
   - 泰勒（F.W. Taylor）和吉尔布雷思（F.B. Gilbreth）开创动作时间研究  
   - 1919年国际劳工组织（ILO）成立，开始保护劳工法案  

2. **第二阶段：1945年~1960年**  
   - 二次世界大战和运筹学时期  
   - 人机匹配研究快速发展，应用于武器设计  

3. **第三阶段：1960年~1980年**  
   - 迅速成长时期，应用于军事工业、航天工业、计算机公司等  

4. **第四阶段：1980年~1990年**  
   - 计算机、灾难和诉讼时期  
   - 重大事故如三哩岛核电站事故、挑战者号航天飞机失事等，强调人因错误的重要性  

5. **第五阶段：1990年以后**  
   - 广泛应用时期，渗透到计算机界面、医疗设施、老年人产品等领域  
   - 关注满意度、幸福感、尊严  

6. **21世纪的人因工程**  
   - 技术的人性化（如操作系统演变）  
   - 人的技术化（提高能力但可能产生依赖）  

### 人因工程学的重要性及应用领域

| 范围                 | 对象举例                     | 例子                       |
| -------------------- | ---------------------------- | -------------------------- |
| 产品和工具设计及改进 | 机电设备、交通工具、建筑设施 | 机床、汽车、城市规划       |
| 作业的设计与改进     | 作业姿势、方法、工具配置     | 工厂生产作业、车辆驾驶作业 |
| 环境的设计与改进     | 声、光、热、色彩等环境因素   | 工厂车间、办公室、驾驶室   |

## Part 2: 人体测量

### 人体测量基础
人体测量（Anthropometry）是指对人类身体各方面特征数据的度量，特别是人体的尺寸（静态和动态）、形状和耐力，以及这些数据在设计中的应用。

**重要性**：设计的前提，确保产品适合人，而不是让人适应产品。

### 测量姿势
- **立姿**：挺胸直立，头部以眼耳平面定位，眼睛平视前方，上肢自然下垂。
- **坐姿**：挺胸坐在平面上，膝弯曲成直角，足平放地面。



### 人体测量仪器
- 人体测高仪
- 人体测量用直脚规
- 人体测量用弯脚规

### 静态测量数据统计描述
人体测量数据近似服从正态分布。常用百分位数表示设计适应域：
- 第5百分位数（P5）：代表“小身材”，5%人群数值低于此值
- 第50百分位数（P50）：代表“适中”身材
- 第95百分位数（P95）：代表“大身材”，5%人群数值高于此值

**百分位数计算公式**：
- 1%~50%： \( P_v = \bar{X} - (S \times K) \)
- 50%~99%： \( P_v = \bar{X} + (S \times K) \)
  - \(\bar{X}\)：平均值
  - \(S\)：标准差
  - \(K\)：百分比变换系数（查表）

**百分比与变换系数K表**：

| 百分比(%) | K     | 百分比(%) | K     | 百分比(%) | K     |
| --------- | ----- | --------- | ----- | --------- | ----- |
| 0.5       | 2.576 | 25        | 0.674 | 90        | 1.282 |
| 5         | 1.645 | 50        | 0.000 | 95        | 1.645 |
| 10        | 1.282 | 75        | 0.674 | 99        | 2.326 |

### 静态测量与动态测量
- **静态测量**：身体保持固定，测量构造尺寸（如身高、臂长）
- **动态测量**：身体移动，测量功能尺寸（如活动空间、伸展域）

**常用人体功能尺寸测量区域示例**：

| 测量项目       | 男(P5/P50/P95) mm | 女(P5/P50/P95) mm |
| -------------- | ----------------- | ----------------- |
| 立姿双手上举高 | 1971/2108/2245    | 1845/1968/2089    |
| 坐姿上肢前伸长 | 777/834/892       | 712/764/818       |



### 人体测量数据的影响因素
- **民族因素**：不同民族有独特的人体数据
- **性别、年龄因素**：男性平均身高比女性高100mm；体形随年龄变化
- **职业因素**：不同职业群体尺寸存在差异



### 人体测量数据在设计中的运用原则
1. **极端设计原则**：满足所有总体，采用最大值和最小值（如安全门高度以P99为依据）
2. **可调设计原则**：设计可调节范围，覆盖P5女性到P95男性（如办公椅高度）
3. **平均设计原则**：用于非关键工作，采用平均数值（如超市结账柜台）

**设计基准点**：
- 驾驶舱设计：以臀部或眼睛为基准点
- 工作台设计：以手或眼为基准点

### 人体模板的应用
人体模板是按标准人体尺寸制成的二维模型，用于辅助设计（如工作台、驾驶室）。



### 案例：自行车设计
自行车设计需考虑的人体尺寸包括车架高度、把手位置等，以人体测量数据为参照，确保舒适性和安全性。

## 总结
人因工程学通过科学的数据和方法，优化人、机、环境之间的交互，提升工作效率、安全性和舒适性。人体测量作为基础工具，为设计提供关键依据，确保产品符合用户需求。


## 人体生理特性与人因设计

### 感觉阈限与感受性
人的各种感受器都有一定的感受性和感觉阈限，刺激必须达到一定强度才能对感受器官发生作用。


### 感觉适应现象
感觉器官经过连续刺激一段时间后，敏感性会降低，产生适应现象。一种感受器官只能接受一种刺激和识别某一种特征。

**感觉信息分布**：
- 眼睛：80%
- 耳朵：14% 
- 其他器官：6%


### 一、视力与视角

**视力定义**：眼睛分辨物体细部的最大能力，以辨别最小视角（临界视角）的倒数为指标。


**视角计算**：

$$\alpha=2arctg(D/2L)$$

- $\alpha$：视角（度）
- D：视看对象两点间距离
- L：眼至视看对象的距离

**简化公式**：可视距离 = 物体的长度 × 57.3 / 视角（度）

**视角与视力关系表**：

| h/d     | 视角(分) | 视力 |
| ------- | -------- | ---- |
| .017455 | 60.00    | 0.02 |
| .000500 | 1.72     | 0.58 |
| .000241 | 1.00     | 1.00 |
| .000050 | 0.17     | 5.81 |


### 二、视野范围

**视野特点**：
- 最佳可视距离：上下10度
- 最近可视距离：上下60度，左右30度
- 不同颜色视野范围不同


**应用案例**：公路交叉口交通标志设置需要考虑视野范围，确保驾驶员能够及时识别。


### 三、视距与作业

**视距定义**：人在作业中正常的观察距离，通常采用380-760mm的视距范围。

**作业精度与视距关系**：


| 作业类型   | 示例              | 视距离(cm) | 固定视野直径(cm) | 作业姿势   |
| ---------- | ----------------- | ---------- | ---------------- | ---------- |
| 最精细作业 | 安装最小部件      | 12-25      | 20-40            | 坐姿       |
| 精细作业   | 安装收音机/电视机 | 25-35      | 40-60            | 坐姿或站姿 |
| 中等粗活   | 印刷/机床操作     | <50        | <80              | 坐姿或站姿 |
| 粗活       | 包装/粗磨         | 50-150     | 30-250           | 坐姿       |
| 远视       | 黑板/开汽车       | >150       | >250             | 坐姿或站姿 |

### 四、明视觉与暗视觉

**视网膜感光机能**：
- **视杆细胞**：对弱光敏感，产生暗视觉（Scotopic vision）
- **视锥细胞**：对强光和色光敏感，产生明视觉（Photopic Vision）和色觉


**适应现象**：
1. **明适应**：由暗处进入明处，约1分钟恢复
2. **暗适应**：由明处进入暗处，约25-30分钟充分适应


**应用案例**：医院手术室照明设计
- 手术台照度：10000-50000lx（可调节）
- 背景照度：400-500lx
- 无影灯要求：20000-100000lx，光束直径10-25cm

**照明设计原则**：
- 亮度比控制：邻近环境3:1，远处暗环境10:1，亮环境1:10
- 避免强反射面引起眩光


### 五、色觉

**颜色对比效果**（从好到差排列）：
蓝白、黑黄、绿白、黑白、绿红、红黄、红白、橙黑、黑绛、橙白

**颜色辨别特性**：


**色彩的心理效应**：
- **暖色调**（红橙黄褐）：温馨、和谐、热情、愉快
- **冷色调**（青绿蓝）：宁静、高雅、清凉

**色彩搭配方法**：
- 同色系颜色搭配
- 相邻色搭配  
- 补色搭配

**应用案例**：图书馆色彩设计
- 公共空间：暖色调，醒目温馨
- 阅览室：绿色为主，明朗宁静，有助于消除疲劳


### 六、视错觉

**主要错觉类型**：

1. **长度错觉**：物体两端增添附加物导致长短判断错误


2. **方位错觉**：物体被隔开或受干扰而歪曲形象


3. **透视错觉**：远小近大效应


4. **对比错觉**：周围物体大小影响判断


5. **大小-重量错觉**：小物体比大物体感觉更重
6. **空间定位错觉**：视觉与平衡觉不协调
7. **颜色错觉**：颜色影响重量感、空间感、温度感、声音感
8. **似动现象**：自动运动、诱导运动、假现运动


9. **形状竞争**：双重知觉现象


### 七、视觉运动规律

1. **运动方向**：水平运动比垂直运动快，视线习惯从左到右、从上到下
2. **观察象限**：左上象限最优，右下象限最差
3. **眼动协调**：两眼运动同步协调
4. **注视时间**：
   - 基本辨认：0.07-0.3s（平均0.17s）
   - 清晰辨认：2-3秒
   - 阅读速度：180-240字/分

**页面显示时间计算**：
页面显示时间 = 清晰辨认目标时间 + 阅读时间
阅读时间 = 目标字个数 / 阅读速度

## 总结

视觉系统的人因设计需要考虑视力、视角、视野、视距、明暗适应、色觉、视错觉和视觉运动规律等多方面因素。合理的设计能够提高工作效率，减少视觉疲劳，确保操作安全。在实际应用中，需要结合具体工作环境和任务要求，进行综合性的视觉工效学设计。
