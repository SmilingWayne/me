    朱兰质量管理三部曲是质量管理的核心框架，强调质量策划、质量控制和质量改进的循环过程。


## QC七大手法（老七）
QC七大手法是质量控制中常用的七种工具，分为定量工具和定性工具：
- **定量工具**：控制图、直方图、排列图、散布图
- **定性工具**：调查表、分层法、因果分析图

### 检查表法

检查表（又称调查表或核对表）用于收集和整理质量原始数据，并进行粗略原因分析。常见类型包括：
- 缺陷位置检查表
- 不合格品项目检查表
- 不合格原因检查表
- 数据分布检查表

**示例表格：预制大梁不良项目调查表**

| 施工工段 | 蜂窝麻面 | 漏筋 | 胀模 | 漏浆 | 表面不平 | 埋件偏差 | 其他 |
| -------- | -------- | ---- | ---- | ---- | -------- | -------- | ---- |
| A        | 1        |      | 7    | 3    | 1        | 2        |      |
| B        |          | 1    | 6    | 2    |          | 2        |      |
| C        |          |      | 5    | 2    |          | 1        |      |



### 分层法
分层法是将质量数据按特定特征（如时间、操作人员、机器设备等）归类整理的统计分析方法。

**示例表格：操作班检查表**

| 操作班 | 合格 | 不合格 | 不合格率 |
| ------ | ---- | ------ | -------- |
| 一     | 15   | 5      | 25%      |
| 二     | 8    | 12     | 60%      |
| 三     | 16   | 4      | 20%      |
| 合计   | 39   | 21     | 35%      |

### 因果图

因果图（鱼刺图）用于分析质量特性与原因之间的关系，作图步骤包括：
1. 明确分析问题
2. 组织讨论
3. **按5M1E（人、机、料、法、环、测）分类找出大原因**
4. 逐层分析中、小原因
5. 识别重要原因
6. 记录相关事项



### 排列图
排列图基于帕累托原理（“关键的少数，次要的多数”），通过累计百分比曲线识别关键问题。因素分类：
- **A类因素**：累积频率0~80%，主要因素
- **B类因素**：累积频率80%~90%，次要因素
- **C类因素**：累积频率90%~100%，一般因素

**制作步骤**：
1. 按频数从大到小排列数据
2. 计算累计频数与频率
3. 绘制直方图和累计曲线
4. 分类并制定改进措施

**示例表格：故障原因排列**

| 序号 | 故障原因 | 故障次数 | 累计频数 | 累计频率 |
| ---- | -------- | -------- | -------- | -------- |
| 1    | 早期故障 | 54       | 54       | 41.9%    |
| 2    | 操作失误 | 35       | 89       | 69.0%    |
| 3    | 耗损故障 | 22       | 111      | 86.0%    |
| 4    | 原因不明 | 18       | 129      | 100%     |



### 散布图
散布图用于分析两个变量之间的相关关系，常见类型包括：
- 正相关（强/弱）
- 负相关（强/弱）
- 不相关
- 非线性相关

**制作步骤**：
1. 确定相关变量
2. 收集至少30组数据
3. 绘制坐标轴（横轴原因，纵轴结果）
4. 标注数据点
5. 观察图形并建立模型
6. 分析关系强度



## 质量改进的工作方法：PDCA循环

PDCA循环（戴明环）包括四个阶段：
- **Plan（计划）**：分析现状，找出问题，制定措施（5W1H）
- **Do（执行）**：实施计划内容
- **Check（检查）**：评估执行效果
- **Action（处理）**：标准化成功经验，解决遗留问题

**特点**：
- 大小循环互动
- 阶梯式提升质量水平



## TQM的有效组织：QC小组活动
QC小组是群众性质量管理活动的组织形式，特点包括：
- **自主性**：员工自愿参与，自我管理
- **群众性**：涵盖领导、技术人员和一线员工
- **民主性**：平等讨论，集思广益
- **科学性**：基于数据分析和统计方法

### 注册登记要求
- 小组规模：3-10人
- 每年登记一次，按课题注册
- 活动前登记，可自定义小组名称

**示例注册表**：

| 项目     | 内容                   |
| -------- | ---------------------- |
| 小组名称 | 日月星辰QC小组         |
| 课题名称 | 降低直拉钢丝亮丝不良率 |
| 课题类型 | 现场型                 |
| 注册时间 | 2013.1.10              |

### QC小组活动程序
1. **选择课题**：基于问题重要性、可行性（如降低不良率）
2. **掌握现状**：收集数据，识别症结（如使用排列图分析）
3. **目标设定**：明确可量化目标（如不良率降至2.6%）
4. **原因分析**：使用因果图找出潜在原因
5. **确定要因**：通过数据验证关键因素
6. **制定对策**：采用5W1H表格规划措施
7. **实施对策**：按计划执行并记录过程
8. **检查效果**：对比目标，评估经济效益和社会效益
9. **制定巩固措施**：标准化有效做法，修改作业文件
10. **总结和下一步打算**：回顾成果，规划新课题



### 统计工具应用
- **简易图表**：趋势图、柱状图、饼图
- **传统质量工具**：检查表、直方图、控制图、排列图
- **数据分析**：确保客观性、可比性、时间性

通过PDCA循环和QC小组活动，实现持续质量改进，提升组织效能。
