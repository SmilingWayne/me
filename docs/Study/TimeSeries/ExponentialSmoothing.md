# 指数平滑法

!!! example "本章概要"

    系统讲解指数平滑法及其扩展模型。从简单指数平滑出发，逐步引入趋势项 (Holt)、季节项 (Holt-Winters)、Damped 修正，最终建立 ETS 状态空间模型框架。**核心洞见**：指数平滑利用所有历史数据但权重指数衰减。通过 (Error, Trend, Seasonal) 三元组划分 18 个模型，使用 AIC/BIC 进行模型选择，支持区间预测。

---

## 1. 简单指数平滑法

在移动平均法中，我们规定了一个固定长度的滑动窗口，将这个窗口内的需求均值（或者加权均值）作为下一个时刻的预测值。这种方法在应用时可能**缺乏灵活度**、无法利用移动平均窗口外的需求数据，同时，预测过程除了实际值外，还伴随有历史数据的预测误差。我们同样希望借助过去"真实值与预测值的误差"，给新的时刻的需求预测提供一些启发。

### 1.1 基本公式

!!! definition "简单指数平滑公式"

    用 $y_t$ 表示 $t$ 时刻的实际需求数据，用 $\hat{y}_{t+1|t}$ 表示 $t$ 时刻对 $t+1$ 时刻需求的预测值。事先给定权重 $\alpha \in [0,1]$：
    $$
    \hat{y}_{t+1|t} = \alpha y_t + (1-\alpha) \hat{y}_{t|t-1}
    $$

**公式含义**：下一个时刻需求的预测值，等于该时刻的实际需求与前一时刻对当前时刻的指数平滑预测值的加权和。

!!! info "核心洞见"

    正因为预测值也包括了前一时刻的预测值，所以**指数平滑实际上利用了所有的历史数据**，只不过先前历史数据的权重会随着时间推移指数下降，变得越来越不重要，而靠近现在的数据会占有更高的权重。

### 1.2 计算示例

对于时间序列 `[10, 20, 40, 20, 30, 30]`，借助简单指数平滑进行预测。将最后一个时刻的数据作为测试集，假定 $\alpha = 0.5$：

| 时刻 | 当前需求 $y_t$ | 上期预测 $\hat{y}_{t | t-1}$                                         | 下期预测 $\hat{y}_{t+1 | t} = 0.5 y_t + 0.5 \hat{y}_{t | t-1}$ |
| ---- | -------------- | -------------------- | --------------------------------------------- |
| 1    | 10             | 无（设为 0）         | $0.5 \times 10 + 0 = 5$                       |
| 2    | 20             | 5                    | $0.5 \times 20 + 0.5 \times 5 = 12.5$         |
| 3    | 40             | 12.5                 | $0.5 \times 40 + 12.5 \times 0.5 = 26.25$     |
| 4    | 20             | 26.25                | $0.5 \times 20 + 26.25 \times 0.5 = 23.125$   |
| 5    | 30             | 23.125               | $0.5 \times 30 + 23.125 \times 0.5 = 26.5625$ |

这个值与真实值（30）的相关评价函数均可对应计算出来。

### 1.3 参数调优

可见，在不考虑多余因素的情况下，指数平滑的计算结果仅和 $\alpha$ 有关。所以通过取不同 $\alpha$，评估预测结果，就可以**选出最优权重**。

## 2. 考虑趋势项：Holt 模型

简单指数平滑方法默认不考虑趋势项和季节项，一般而言称为**"水平项"预测**。如果考虑趋势项，指数平滑的公式会有所不同。这种模型也被称为**Holt 模型**（Holt's Linear Trend Method）。

### 2.1 Holt 模型公式

下一个时刻的需求，是由**水平项 $l_t$** 和**趋势项 $b_t$** 共同决定：

$$
\hat{y}_{t+1|t} = l_t + b_t
$$

其中：

| 项               | 公式                                                 | 含义                                       |
| ---------------- | ---------------------------------------------------- | ------------------------------------------ |
| **水平项** $l_t$ | $l_t = \alpha y_t + (1 - \alpha)(l_{t-1} + b_{t-1})$ | 同时考虑实际需求、前一时期的预测值、趋势项 |
| **趋势项** $b_t$ | $b_t = \beta(l_t - l_{t-1}) + (1 - \beta)b_{t-1}$    | 水平项在 $t$ 时刻的"斜率"                  |

!!! info "与简单指数平滑的区别"

    相比于简单的指数平滑，水平项不只是由实际需求与前一个时期的预测值来决定，而是**同时考虑实际需求、前一个时期的预测值、趋势项**。

### 2.2 参数说明

| 参数     | 名称           | 含义                 |
| -------- | -------------- | -------------------- |
| $\alpha$ | 水平项平滑参数 | 控制水平项的更新速度 |
| $\beta$  | 趋势项平滑参数 | 控制趋势项的更新速度 |

与简单指数平滑不同，我们需要求出**2 个不同的参数**。

### 2.3 多步前向预测

如果借助趋势项做 $h$ 步前向预测，那么公式变为：

$$
\hat{y}_{t+h|t} = l_t + h b_t
$$

也就是说，对未来进行多步预测，结果实际上是：**当前预测的下一时刻的水平项 + 趋势项与步数的乘积**，是一个与 $h$ 有关的线性函数。

## 3. Damped 趋势修正

### 3.1 问题背景

上述处理趋势的手段，如果进行多步预测，则 $b_t$ 都是相同的。**实证数据发现**：如果预测步长较长，这种方式会**高估实际需求**。

### 3.2 Damped 方法

Gardner & McKenzie (1985) 指出可以通过在趋势项上增加额外系数 $\phi$，让趋势项在一定步长后近似被拉平（Flat），一些实际数据中能获得较好的效果。

!!! info "示意图"

    > [Image] - Damped 趋势效果对比图

!!! quote "Gardner & McKenzie (1985)"

    > Gardner & McKenzie introduced a parameter that "dampens" the trend to a flat line some time in the future. Methods that include a damped trend have proven to be very successful, and are arguably the most popular individual methods when forecasts are required automatically for many series.

### 3.3 公式理解角度

- 在没有引入 $\phi$ 之前，对 $h$ 步未来预测，$b_t$ 前的系数是正整数 $h$
- 现在变成一个长度为 $h$、首项为 $\phi$、公比为 $\phi < 1$ 的等比数列的和
- 随着 $\phi$ 的增加，数列和的增加速度会迅速下降，近似于拉平了趋势项

!!! warning "$\phi$ 参数取值"

    实际上，$\phi$ 取值对于较短的多步预测是很敏感的：
    - 取得太小：第一个预测值的趋势项的重要性会被低估
    - 取得太大：失去 Flatten 的效果

    根据经验，**取值应在 [0.8, 0.98] 之间**。

## 4. 同时考虑趋势项和季节项：Holt-Winters 模型

又被称为**Holt-Winters 模型**，它将下一个时刻的需求视为如下三个部分的和：

1. **当前时刻的实际需求（水平项）** $l_t$
2. **趋势项** $b_t$
3. **要预测的时刻的季节项** $s_{t+h-m(k+1)}$

### 4.1 参数说明

理解一下这里的 $k, m$：

| 符号 | 含义                       | 示例                            |
| ---- | -------------------------- | ------------------------------- |
| $m$  | 季节长度                   | 季度数据 $m=4$，月度数据 $m=12$ |
| $k$  | $\frac{h-1}{m}$ 的整数部分 | 用于定位对应的季节项            |

${t+h-m(k+1)}$ 的作用就是限制了要预测时刻的季节项等于上一个已知对应季节项。

!!! example "示例"

    $t = 100, h = 22, m = 5$。我们要预测第 122 时刻。此时该时刻对应的季节：
    
    $$
    122 - 5 \times (4 + 1) = 97
    $$
    
    正好就是已知时刻里，最近的对应的季节项。

### 4.2 Holt-Winters 加法模型公式

整个公式写成：

$$
\hat{y}_{t+h|t} = l_t + b_t + s_{t + h - m(k+1)}
$$

其中：

| 项               | 公式                                                               | 含义                                                                                   |
| ---------------- | ------------------------------------------------------------------ | -------------------------------------------------------------------------------------- |
| **水平项** $l_t$ | $l_t = \alpha (y_t - s_{t - m}) + (1 - \alpha)(l_{t-1} + b_{t-1})$ | 当前时刻减去季节项后的真实需求，与前一个时期水平项值与趋势项之和的指数平滑             |
| **趋势项** $b_t$ | $b_t = \beta (l_t - l_{t-1}) + (1-\beta)b_{t-1}$                   | 当前时刻水平项的变化值与前一个时期趋势项的指数平滑                                     |
| **季节项** $s_t$ | $s_t = \gamma (y_t - l_{t-1} - b_{t-1}) + (1 - \gamma) s_{t - m}$  | 当前时刻减去趋势项和水平项后由季节项决定的那部分需求，与上一个周期对应季节项的指数平滑 |

## 5. 加法模型与乘法模型

同时考虑趋势项和季节项目的模型中，各个组成部分可以通过**加法**或**乘法**组合。

### 5.1 加法模型

$$
\hat{y}_{t+h|t} = l_t + b_t + s_{t + h - m(k+1)}
$$

### 5.2 乘法模型

将季节项作为一个乘数，乘到水平项 + 趋势项上：

$$
\hat{y}_{t+h|t} = (l_t + b_t) \cdot s_{t+h-m(k+1)}
$$

!!! info "关键区别"

    两种模型都是刻画趋势项和季节项的方法。但是需要注意：**加法模型和乘法模型改变/操作的对象都是季节项，与趋势项是无关的**。

!!! info "示意图"

    > [Image] - 加法模型示例
    > [Image] - 乘法模型示例

### 5.3 Damped 方法（季节项模型中）

与前面提到的方法一样，Damped 方法是应对线性趋势项在长步长预测中后期会夸大趋势项水平的情况提出的解决方案。解决方法依然是添加一个指数项，对步长较长的部分进行修正。

## 6. 指数平滑模型的分类框架

到目前为止，上面的内容已经讨论了几个因素：

| 因素                          | 选项           |
| ----------------------------- | -------------- |
| 1. 简单指数平滑               | 基础模型       |
| 2. 考虑季节项的 ES            | 加法/乘法      |
| 3. 考虑趋势项的 ES            | 无/加法/Damped |
| 4. 趋势项是否考虑 Damped 修正 | 是/否          |
| 5. 季节项采取加法 or 乘法模型 | 加法/乘法      |

有没有一种办法，可以灵活地选取某个最好的指数平滑的组合，对未来进行预测呢？

## 7. 状态空间模型 (State Space Models)

在前面所有的模型中，误差都是通过加减法计算得来的，也就是 $e_t = y_t - \hat{y}_{t|t-1}$。而实际上，误差也可以通过乘法法则计算。

为了和季节项中的乘法和加法模型进行区分，需要引入一个新的指标。加上前一节提出的对趋势项、季节项的指标，可以组成一个**三元组**，对模型进行划分：**$(E, T, S)$**。

| 维度                | 标记  | 含义                      |
| ------------------- | ----- | ------------------------- |
| **Error (误差)**    | $A$   | 加法误差 (Additive)       |
|                     | $M$   | 乘法误差 (Multiplicative) |
| **Trend (趋势)**    | $N$   | 无趋势                    |
|                     | $A$   | 加法趋势                  |
|                     | $A_d$ | Damped 趋势               |
| **Seasonal (季节)** | $N$   | 无季节                    |
|                     | $A$   | 加法季节                  |
|                     | $M$   | 乘法季节                  |

### 7.1 ETS(A,N,N)：考虑加法误差的简单指数平滑

用简单的指数平滑做示例：

!!! info "示意图"

    > [Image] - ETS(A,N,N) 示意图

该时刻的水平项 = 上一时刻的水平项 + $\alpha e_t$。

如何对残差进行分析？此时我们假定误差 $e_t$ 服从 $N(0, \sigma^2)$ 的正态分布 $\varepsilon_t$，是一个白噪声。所以有：

$$
\begin{aligned}
y_t &= l_{t-1} + \varepsilon_t \tag{1.1} \\
l_t &= l_{t-1} + \alpha \varepsilon_t \tag{1.2}
\end{aligned}
$$

| 方程  | 名称                     | 含义                                                                                                                                   |
| ----- | ------------------------ | -------------------------------------------------------------------------------------------------------------------------------------- |
| (1.1) | **Measurement Equation** | 将真实值表示为预测值 $l_{t-1}$（对应于真实需求中可以被预测的那部分）与误差 $\varepsilon_t$（对应真实需求中没有被预测的部分）的线性函数 |
| (1.2) | **State Equation**       | 表示预测过程中状态的转换，$\alpha$ 可以理解为前后两个需求水平变化的幅度                                                                |

!!! quote "State Space Models 定义"

    > Each model consists of a measurement equation that describes the observed data, and some state equations that describe how the unobserved components or states (level, trend, seasonal) change over time. Hence, these are referred to as state space models.

### 7.2 ETS(M,N,N)：考虑乘法误差的简单指数平滑

乘法误差是如何刻画的？

!!! info "示意图"

    > [Image] - 乘法误差示意图

也就是用 **(真实值 - 预测值) / 预测值** 来反映误差的情况。此时假定 $\varepsilon_t$ 依然是均值为 0 的正态分布。

对应的，我们有：

$$
y_t = l_{t-1} + l_{t-1} \varepsilon_t \tag{1.3}
$$

可以看到真实值是通过预测值与预测值和误差的乘积两个数之和计算出来的。所以可以写出如下的 Measurement Equation 和 State Equation：

$$
\begin{aligned}
y_t &= l_{t-1} (1 + \varepsilon_t) \tag{1.4} \\
l_t &= l_{t-1} (1 + \alpha \varepsilon_t) \tag{1.5}
\end{aligned}
$$

!!! info "核心思想"

    上述的重点在于，**将概率分布的概念（而不只是误差）融入到了时间序列的计算中**。

### 7.3 18 个 ETS 模型

同样地，我们通过 3 个参数：

- $(A, M)$ - 误差项
- $(N, A, A_d)$ - 趋势项
- $(N, A, M)$ - 季节项

利用笛卡尔积，可以区分出**18 个不同的模型**：

!!! info "18 个 ETS 模型表"

    > [Image] - 18 个 ETS 模型分类表

!!! warning "参数限制"

    值得注意的是，在一些模型中，尤其是季节因素考虑乘法模型时，模型参数的取值范围有限制，比如 $0 < \beta < \alpha$ 等。

### 7.4 ETS 模型的参数估计

对于误差为加法模型，最大化概率与最小均方误差的结果是一样的；但是对于误差乘法的模型，结果会不同。

简而言之，就是对以下内容进行估计：

- 平滑参数：$\alpha, \beta, \gamma, \phi$
- 初始状态：$l_0, b_0, s_0, s_{-1}$ 等

在点估计模型中，我们调参的手段是通过不同的参数组合，求出使得样本内测试集上 MSE 最小的那个当作最优参数。但是当误差变成正态分布之后，无法利用之前相同的方法进行估计了。我们转而使用**极大似然估计法**：

!!! goal "极大似然估计"

    找到具有某个误差概率分布的模型，使得真实的需求值产生自该模型的概率最大。

!!! warning "不稳定的模型"

    书中并没有对调参的过程给出足够详细的解释，但是指出了针对 3 个模型：**ETS(A, N, M)**、**ETS(A, A, M)**、**ETS(A, A_d, M)**，他们在状态方程上可能出现分母过小接近 0 的状态，在选择模型的时候一般不予考虑。

!!! quote "乘法误差模型的适用条件"

    > Models with multiplicative errors are useful when the data are strictly positive, but are not numerically stable when the data contain zeros or negative values. Therefore, multiplicative error models will not be considered if the time series is not strictly positive. In that case, only the six fully additive models will be applied.

### 7.5 模型选择：AIC/BIC 准则

对于多个 ETS 模型，基于给定的时间序列，借助**AIC / BIC 准则**选择最适合的模型组合 $(E, T, S)$。

| 准则                                     | 公式                         | 说明                                                                                                                                                                   |
| ---------------------------------------- | ---------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **AIC** (Akaike's Information Criterion) | $AIC = -2 \log(L) + 2k$      | $L$ 是历史数据来自于给定的某个模型对应的参数组合的概率（一般通过极大似然估计给出）；$k$ 是已估计的所有参数和初始状态参数的数量（对应统计学中的 independent variables） |
| **BIC** (Bayesian Information Criterion) | $BIC = AIC + k[\log(T) - 2]$ | $T$ 是样本量                                                                                                                                                           |

## 8. 基于 ETS 进行区间预测

对于大部分 ETS 模型，可以把预测区间写作如下格式：

$$
\hat{y}_{T + h | T} \pm c \cdot \sigma_h
$$

### 8.1 Coverage 乘数 $c$

这里的 $c$ 依赖于指定的 Coverage Possibility 给出的乘数，也就是为了满足置信水平，在标准正态分布下自变量对应的取值的上下界：

!!! info "标准正态分布临界值表"

    > [Image] - Coverage 乘数表
    > [Image] - 置信水平对应表

### 8.2 预测标准差 $\sigma_h$

公式里的 $\sigma_h$ 指的是**预测的标准差**（Forecast Variance）。这个值的推导的过程非常复杂，加法部分的推导可以参考：

!!! info "方差推导"

可以理解为：**对未来多个时间的预测的方差，等于最优模型相关参数与残差的标准差之间的一个解析式**。

## 9. ETS 和指数平滑的区别与共性

### 9.1 区别

| 方面         | 经典指数平滑   | ETS 状态空间模型               |
| ------------ | -------------- | ------------------------------ |
| **残差处理** | 只考虑加法误差 | 同时考虑加法和乘法模式         |
| **预测类型** | 点预测         | 可进行区间预测（结合概率分布） |
| **模型比较** | 使用 MSE       | 使用 AIC/BIC                   |

### 9.2 共性

| 共性         | 说明                                           |
| ------------ | ---------------------------------------------- |
| **考虑因素** | 均包含了趋势（是否 Damped）、季节（加法/乘法） |
