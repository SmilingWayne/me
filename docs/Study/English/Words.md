# 记一下各种奇怪的英语单词

- `In` 短语家族
  - in short
  - in essence
  - in case 
  - in turn
  - in return
  - in response to: 对某事的反应；to之后的内容促使前面的内容发生
  - in reply to
  - in a sense:
  - in effect:
