## 杂七杂八

- 大括号：
$max \left\{ \frac{pV}{nrT} \right\}$

``` LaTeX 
 $max \left\{ \frac{pV}{nrT} \right\}$
```

- 在min/max的正下方打出下标：
  
$\mathop{\arg\min}\limits_{\theta}$ 以及 ：$\mathop{\min}\limits_{\theta}$

``` LaTeX

$\mathop{\arg\min}\limits_{\theta}$

$\mathop{\min}\limits_{\theta}$

```

- 累乘符号：
  $\prod \limits_{i=0}^n$

``` LaTeX
$\prod \limits_{i=0}^n$
```
- 在求和符号正上方和正下方加字母:
  
  $\sum \limits_{i=1}^{n}$

```LaTeX
$\sum \limits_{i=1}^{n}$
```
- 在箭头正上方和正下方加字符：

$A\stackrel{r/c/}{\rightarrow}B$

```LaTeX
$A\stackrel{r/c/}{\rightarrow}B$
```
