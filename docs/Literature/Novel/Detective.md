# 推理小说｜侦探故事

## 砂之器

!!! note "砂之器"
    松本清张的作品。早期日本推理小说。故事连贯性不错，虽然找出线索的部分有些“brute force”和刻意的巧合，但是可能也是作者不得已为之，毕竟其风格已经开天辟地了，环环相扣很紧凑。

    核心是“子不认父”。

- 所谓人生，常常会因为一个小小的机遇而改变命运。

!!! quote "孤独的孩子，在寂静的河边堆制砂器 / 有不期而至的大雨 / 在砂器初成之际。"

## 占星术杀人魔法

岛田庄司的代表作吧？阅读体验大致如下：

一开始，喔唷！玄之又玄，有星座有神秘学，还有勾人心弦的恐怖作案手法，这下好玩了！

看着看着，啧，有点啰嗦，特别是男主自己去探案那段，深感可以压缩，因为并没有提供什么信息，而且人物塑造上的“我”也绝非主线。

然后看到作者自己跳出来写了一封挑战信，我当场就不服气了，感觉这是他故意在使绊子，于是特地去前面捋了一下情节，既然：

1. 作者强调了很多很多遍“没有作案情节”；
2. 受害者来源涉及到两任妻子 + 丈夫和女儿，接连两起案件；
3. 其他铺垫

我就觉得最重要的突破口一定来自于：**尸体被处理的情况本身**。而事实上最终的结局也类似。

只不过我以为岛田庄司会在最后整个大活，烘托铺垫一大波作案动机，事实证明我想太多。不论如何，这个收尾也足够不错了，一个挺“古典”的动机。

但是这本书依然是不错的。

## 恶意

东野圭吾的作品，我小时候一定在知音还是读者这种杂志上看过这个故事。小说以叙事手法见长，“把一个精彩的故事讲得精彩”，是一件很有挑战性的事。更深刻的是对人物内心动机的反映。

**嫉妒**和**无来由**的恶意。
