# 俄国文学史

本笔记的是俄国文学史家米尔斯基（1890-1939）的《俄国文学史》笔记与一些拓展。

!!! quote "关于作者、《俄国文学史》"
    米尔斯基竟然能成为三种制度的食客：沙皇制度下的公爵、资本主义制度下的教授、共产主义制度下的文人。

    俄国人固有的弥赛亚意识、俄国文学精英的文学优越感、由于深谙西欧文化获得的一种自信，给了本书一种指点江山的豪气与舍我其谁的霸气。


## 古代俄国文学（11～17世纪）

古代俄国的文学语言是教会斯拉夫语。15世纪末，莫斯科衙门的语言又成了官方语言，不过十分简陋。主要内容是一些经书、祈祷书、口头传说、编年史、宗教文本。10～13世纪鞑靼人的入侵，14～15世纪文学的黑暗世纪、宗教绘画兴盛。彼时以基辅为中心，莫斯科公国。

文本
:   《伊戈尔远征记》，1795年发现并后续出版。民族经典。武士贵族精神与开阔的爱国主义。

历史事件
:   ==库里科沃大捷==：1380，弗拉基米尔和莫斯科大公伊万诺维奇率领罗斯诸公国联军与蒙古金帐汗国军队在库里科沃原野展开的中世纪大规模会战，最终击败金帐汗入侵的历史事件。

    沙皇阿历克塞时期的==教会大分裂==（17C40s）、大司祭阿瓦库姆。
    
    彼得大帝的改革。布道词。


## 古典主义时期

### 罗蒙诺索夫 (1711~1765)

捕鱼人的儿子，曾出国学习。怀有两种激情：爱国主义和对科学的爱。也是物理学家、化学家。

==文学上制定文学语言的标准==，引入新的作诗法体系，为俄国语言引入了标准。“古典主义性质的话语风格学说”。

后继者包括苏马洛科夫（1717～1774），家庭殷实的莫斯科上流人士，熟知法语文学。是俄国古典主义朝代的开创者。

### 杰尔查文 (1743~1816)

小贵族家庭出身。“一种野蛮人的古典主义”，哲学上是一种热切欢乐的伊壁鸠鲁主义者，本质上是一种“鞑靼人的俄语”（by 普希金）。主要写宗教颂诗，具有东正教风格的诗歌创作、仪典颂诗。

### 卡拉姆津 (1766～1826)

卡拉姆津出生于外国贵族家庭，深入了解西欧文学，倡导文学语言革命（卡拉姆津运动）。让俄语和斯拉夫语、拉丁语拉开距离，而更加贴近西欧法语等“有教养的语言”。文学上靠近新感伤派、卢梭。

《一位俄国旅行者的书简》、《俄国国家史》

同时代者：西什科夫元帅 (1754~1843)，保守派、民族主义者，反拿破仑入侵宣言。西什科夫派与卡拉姆津派的对立。

同时代的戏剧也有了发展：模仿莎士比亚等。


### 克雷洛夫 (1769~1844)

父亲是穷军官，小时教育粗浅。写作寓言故事（包括对拉封丹寓言的改写），后来俄国文学主要特征现实主义的萌芽。《克雷洛夫寓言》

## 诗歌的黄金时代

诗歌是卡拉姆津运动的继续，“法国式”的，但是对法国浪漫主义怀有敌意。

### 茹科夫斯基 (1783~1852)

一个乡村贵族的私生子。当过兵，给沙皇亚历山大二世当过老师，翻译家，后来长居德国。1820年之前文学运动领袖。浪漫主义作家。

后继者包括巴丘什科夫，语言风格的现代派、教会斯拉夫语的敌人。

### 普希金 (1799~1837)

父亲是贵族。早年受到法国式文化的影响，后来进入政府部门，隐居，错过十二月党人起义。与官僚机构关系颇深，后决斗失败而亡。

文学上法国式的、古典主义风格的，到后来拜伦式的、浪漫主义风格的。不过在抒情上跳出了拜伦的情感痕迹，形成自己风格。

==《叶甫盖尼·奥涅金》==《青铜骑士》《波尔塔瓦》《茨冈人》，奥涅金成为“俄国小说人物形象的祖先”，1830年后风格变成“模拟化”。 (stylized?)并且开始主要创作散文，但是成就逊于诗歌。《大尉的女儿》《黑桃皇后》

对于斯拉夫派而言，他还不够俄国化，对于激进的西方派而言，他又不够现代。

在亚历山大一世时期，法语对俄国的影响力达到顶点，“我们高傲的寓言至今仍不适合写信”——普希金。在这一时期成长起来的每一个受过教育的贵族，其法语均和俄语一样流利。

同时代的一些诗人还包括：

- 雷列耶夫 (1795~1826)，十二月党人的主要领导者。

- 巴拉丁斯基 (1800~1844)，少年时期因为偷窃被开除，后从军，诗人，《埃达》，《舞会》。古典主义、半浪漫主义者。

- 雅济科夫 (1803~1846)

- 玄学派诗人 格林卡 (1788~1880)，神秘主义者；

### 期刊报业的发展

## 果戈理时代

首先是诗歌的衰败与短暂后退。

### 丘特切夫 (1803~1873)

出生于外省贵族家庭，在外交部门工作并把慕尼黑视为家乡。后来的书刊检察官。坚定的保守派、热烈的泛斯拉夫主义者、善恶二元论者。他的写作与说话均只用法语，他的两任妻子都不会说俄语。只在写诗的时候用俄语。

风格上，雄辩的古典主义与形象化的浪漫主义风格。《海上之梦》《意大利村庄》

### 莱蒙托夫 (1814～1841)

出生于莫斯科军官、小地主家庭。是一个苏格兰人的后代。近卫士官学校。拜伦式的姿态，愤世嫉俗、趾高气扬，<u>寻欢作乐的性爱、浪漫主义的爱情、逢场作戏的唐璜气息</u>是他的主要情感。最后死于决斗，年27岁。

奥涅金诗体，《天使》《萨什卡》《恶魔》《童僧》。

身上具有成为现实主义者的一切气质，被认为是《青铜骑士》到《战争与和平》之间的过渡。

### 果戈理 (1809~1852)

生于乌克兰贵族家庭，父亲是一个小地主和剧作者。果戈理中学时就开始写作，有一种阴郁神秘的风格。后来成为圣彼得堡大学教授。俄国现实主义的奠基者。

> 果戈理的作品不是客观的讽刺，而是主观的讽刺，其人物是对内心活动的反省式的讽刺描摹。==他相信其神圣使命是在道德上复活罪孽深重的俄国==，想要成为一个先知和一个导师。
>
> 他的人物均为内心体验的外在化、客观化的象征。
>
> 他眼中的现实并非现实本身，实际上他很少关注自在的现实，完全凭借想象来塑造人物，但是他确实是一个现实主义作家，<u>因为他将文学中从未拥有的众多现实成分和现实层面引入了文学</u>。

《钦差大臣》《死魂灵》《维伊》《密尔格拉得》

### 同时代的其他文学小组

恰达耶夫 (1793~1856)，《哲学书简》，普希金的友人。

赫尔岑小组（空想社会主义小组）、斯坦凯维奇小组（唯心主义者小组）；莫斯科大学。

斯拉夫派：保守的无政府主义。霍米亚科夫 (1804~1860)，

### 别林斯基 (1810~1848)

父亲是贫穷的军医，“他接受教育的方法与其说是规律的学习，不如说是杂食性的阅读和与大学生们的接触”。

与那杰日金共事时候，别林斯基迷恋谢林的浪漫唯心主义及其创作理念，后来转向巴枯宁、费希特的道德唯心主义，后来走向黑格尔。

《望远镜》杂志撰稿者；《文学的幻想》；

!!! quote "别林斯基的历史意义"
    - 贵族阶级作用的终结：超越阶级的平民知识分子获得了文化领导：杂志报刊；
    - 体现了知识分子的一贯精神：改变世界、社会理想主义；
    - 缺陷也很明显：（米尔斯基的观点），别林斯基应该为19世纪下半时期俄罗斯新闻文体的一片混乱负责。
    - 他的文学教义很难让人接受：他的思想表达之热烈毒害了俄国文学，广泛传播的浪漫主义批评的一切共识（灵感、真诚、天赋和人才），对工作和技巧的蔑视，在很大程度上应该为那种鄙视形式和手艺的态度负责。

革命民主主义者、哲学家、文学评论家。通过他的著作宣传了革命民主主义的政治纲领，而且第一个系统地总结了俄国文学发展的历史，科学地阐述了艺术创作的规律，提出了一系列重要的文学和美学见解。

## 现实主义：小说家：1

俄国现实主义诞生于40年代下半，1846年左右。是果戈理讽刺性自然主义和早些的感伤现实主义（乔治桑）二者的混血儿。

一个特征是艺术上的简洁。

### 陀思妥耶夫斯基 (1821~1881)

父亲是莫斯科一所医院的医生，比较富裕，但是不是农奴主。

《穷人》《地下室手记》《卡拉马佐夫兄弟》《罪与罚》《群魔》《白痴》。

处女作《穷人》大受欢迎（涅克拉索夫、别林斯基），但是后来的作品并不讨这两人喜欢。

### 阿科萨科夫 (1791~)

地主家庭出生，母亲是一个典型的卢梭式教育下成长起来的人。阿科萨科夫后来进入政府机关，书报检查部门。

主要创作特征是客观性、开放性。《家庭纪事》

### 冈查洛夫 (1812~1891)

出生于商人家庭。《癫痫》《因祸得福》《平凡的故事》，描写地主少爷转变为实业家的历程 《战舰巴拉达号》 1859年代表作《奥勃洛摩夫》出版，塑造俄国文学“多余的人”形象 。曾任沙皇政府书报检查官、出版事业委员会委员。


### 屠格涅夫 (1818~1883)

出生于贫穷地主家庭。莫斯科大学。柏林大学：黑格尔的圣殿。

《猎人笔记》《父与子》《木木》《烟》《处女地》

!!! note "《猎人笔记》"
    是一个伟大的文学事件，也是一个伟大的社会事件。所再现的农奴并非一般的人，他们在人性方面始终高于其主人。

每个人都相信屠格涅夫笔下的俄国具有照相般的真实。他塑造过众多优美女性形象，却未能刻画出一位俄国男性英雄。

19世纪50至70年代是屠格涅夫创作的旺盛时期，从19世纪60年代起，屠格涅夫大部分时间在西欧度过，结交了许多作家、艺术家，如左拉、莫泊桑、都德、龚古尔。是最早发现青年莫泊桑文学天赋的人之一。

他的创作步莱蒙托夫的后尘，从莱蒙托夫处取得浪漫主义光环。《猎人笔记》中试图摆脱浪漫主义定式，放弃叙事的骨架，集中描写小块生活。

他的语言是一种精挑细选的语言，十分丰富，却又绝妙地回避那些令读者不快的语句。聚焦农奴制改革时代 (1856~1861)、改革前的反动时期等。


### 皮谢姆斯基 (1820~)

家庭是贵族不过比较贫穷。莫斯科大学。乔治桑的崇拜者。强烈的俄国情感、对外国的东西不感兴趣，不具有斯拉夫派的民族理想主义。

《一千个农奴》《苦命》《贵族领地》《浑浊的海》
