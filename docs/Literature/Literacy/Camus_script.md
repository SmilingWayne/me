# 加缪手记

!!! note
    加缪的这些破碎的文字是我看的很多作家里最吸引我的。不光是他的创作的心路历程，也有他阅读和体验的总结，是夹杂着日记属性、读书笔记属性的作品。有的时候还能看到加缪对其他作者的评价。内容很丰富，五花八门，得以一窥一个伟大的创作者。
