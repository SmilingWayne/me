# 春秋战国文、典故、意象与文学故事

<span style="color:#1E90FF;font-weight:bold">白虹贯日、苍鹰击殿</span>

异常天象，比喻大事发生。语出《战国策・魏策・唐雎不辱使命》。

白虹贯日：

人物：**聂政**（韩国轵人，因杀人避仇至齐国，以屠宰为生）、**严遂**（韩卿，与韩相韩傀结怨）、**韩傀**（韩国相国）

经过：严遂以黄金百镒为聂政母亲祝寿，求其刺杀韩傀。<u>聂政因母亲健在拒绝，待母亲去世、姐姐出嫁后，孤身仗剑闯入韩傀府，刺杀韩傀后自毁面容、挖眼、剖腹而死，防止连累他人。</u> 据说行刺时候“白虹贯日”。

苍鹰击殿：

人物：**要离**（吴国刺客）、**庆忌**（吴王僚之子，武艺高强）、吴王阖闾（公子光，杀僚夺位）

经过：阖闾即位后，庆忌逃往卫国，招募士兵欲复仇。要离自请刺杀庆忌，采用苦肉计：让阖闾斩断其右臂、烧死其妻子，伪装成被追杀的逃犯投奔庆忌。庆忌信任要离，让其陪乘战船。要离趁庆忌不备，用短矛刺中其心窝。庆忌临死前赞叹要离是 "天下勇士"，下令放其回国。

柳下惠，春秋鲁国大夫，因食邑柳下，谥号是惠，所以称柳下惠。他任士师时，因正直三次遭罢免。

<span style="color:#1E90FF;font-weight:bold">“绝缨”</span>：据《说苑》记载，楚庄王与群臣饮酒，灯灭，一人拉了下楚庄王美人的衣服，美人将那人的冠缨拔了下来，让庄王追究。庄王反命暂不掌灯，让每个人都将帽缨摘下，尽欢而散。后来在一次与晋人的作战中，那人表现得非常勇敢。

## 地理

<span style="color:#1E90FF;font-weight:bold">郑渠、白渠</span>：郑渠是战国时水工郑国为秦国所凿，白渠是汉武帝时白公建议开凿，引泾水灌溉关中平原。

<span style="color:#1E90FF;font-weight:bold">玄灞、素浐</span>：长安附近两个水名。灞水的水黑，故云玄灞；浐水的水白，故云素浐。二者均为渭水的支流。后来的“霸陵伤别”的霸陵，是西汉文帝刘恒的陵寝，临近灞水。

<span style="color:#1E90FF;font-weight:bold">崤函</span>，崤：崤山，也作崤陵。
在今**河南三门峡**境内，分东崤、西崤，秦岭东段支脉，山道狭窄险峻，自古为险要山地隘路。函：函谷关。古函谷关在今**河南灵宝**，扼守崤山要道；是战国、秦汉天下第一雄关。

<span style="color:#1E90FF;font-weight:bold">河洛</span>，**河：专指黄河**，**洛：专指洛水（洛河）**，黄河中游以南、洛水流域，核心就是今洛阳盆地一带，夏、商、周三代多建都于河洛，古文里常以「河洛」代指洛阳、中原王畿，文化上代指华夏正统文脉根源。
