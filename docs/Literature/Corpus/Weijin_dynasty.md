# 魏晋南北朝文、典故、意象与文学故事

## 典故

<span style="color:#1E90FF;font-weight:bold">五马渡江</span>

永嘉之乱后，晋朝宗室中，只有琅琊王、汝南王、西阳王、南顿王、古城王同至江东。其中琅琊王司马睿继承帝位，是为晋元帝。

<span style="color:#1E90FF;font-weight:bold">萧娘</span>

女子的泛称。南朝以来，男子所恋的女子常被称为**萧娘**，女子所恋的男子常被称为萧郎。

梁临川王**萧宏**，武帝萧衍的弟弟，文帝萧简之的第六个儿子。萧宏所统领的军队都器械精良，军容强盛，军队驻扎在洛口，前锋攻下梁城。**萧宏不懂军事，指挥调度缺乏常识**，多与朝廷制度不合。诸将想乘胜深入敌境，萧宏听说魏国援军离此不远，**畏缩胆法不敢进军**。魏人笑他怯懦如女子。

<span style="color:#1E90FF;font-weight:bold">偷香</span>

窃香私语，指男女幽会偷情之事。据《晋书•贾充传》，**韩寿是贾充的属官，他姿容美丽**，贾充女爱而与之私通。充女用外国所进香料，衣着人衣，经月不息。**韩寿沾此香气，被贾充发觉**，遂嫁女于寿。 ("韩寿美姿貌，贾充女见而悦之，潜通音好，时西城贡奇香，一者人则经月不歇。帝惟赐充。充女密窃而私贻寿")

后便以“纷香”来指代男女偷情之事。

<span style="color:#1E90FF;font-weight:bold">锦字书 / 回文书</span>

**窦滔**：字连波，扶风郡（今陕西**扶风**）人，前秦苻坚时期的秦州刺史（治所今甘肃天水），出身名门，为右将军窦于真之孙、窦朗之次子，风神秀伟，精通经史，文武双全。

<span style="color:red;font-weight:bold">苏蕙</span>：字若兰，始平郡（今陕西武功）人，陈留令苏道质第三女，**前秦著名才女**，<span style="color:red;font-weight:bold">魏晋三大才女之一（与蔡文姬、谢道韫并称）</span>，识知精明，仪容秀丽，年十六嫁与窦滔，夫妻感情甚笃。

苻坚时期，窦滔是秦州刺史，被流放西去。其妻苏若兰曾于锦上织**回文诗**《璇玑图》寄给被流放的丈夫，倾诉思念之情。史料载诗可正读、反读、纵横读、交叉读、蛇行读、退一字读、左右旋读等，能形成三、四、五、六、七言多种诗体，主题均不离闺怨与思念。

后世用作妻子致书丈夫的典故。 

郑璞

## 书籍、文章

《洛阳伽蓝记》《与山巨源绝交书》

《酒德颂》：刘伶，“无私无虑，其乐陶陶。兀然而醉，豁尔而醒。静听不闻雷霆之声，熟视不睹泰山之形，不觉寒暑之切肌，利欲之感情。俯观万物扰扰焉，如江汉之载浮萍。”

《大人先生传》： 阮籍，**大人先生指能包容天地、施恩百姓、不要名利、先知先觉的人**。原型是当时的隐士孙登。
