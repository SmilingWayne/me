# 安提戈涅（Antigone）的悲剧：底比斯循环

## 禁葬令｜城邦法 vs 神圣/家族义务”

《安提戈涅》的时间点在“俄狄浦斯退出底比斯舞台”之后：他的两个儿子**埃忒俄克勒斯与波吕尼刻斯为王位开战并同归于尽**。([en.wikipedia.org](https://en.wikipedia.org/wiki/Antigone_%28Sophocles_play%29?utm_source=openai))  

克瑞翁继位后宣布：
- 忠于城邦的埃忒俄克勒斯应受礼葬；
- 攻城的波吕尼刻斯视为叛徒，尸体不得埋葬（这在古希腊观念中是极重的羞辱与宗教性灾难）。([britannica.com](https://www.britannica.com/biography/Sophocles/The-plays?utm_source=openai))  

安提戈涅选择违抗：
- 她并非“不懂政治”，而是坚持：**对死者的葬礼义务属于更高层级的规范（神的法/不成文法/家族义务）**，不应被君王命令取消。([britannica.com](https://www.britannica.com/biography/Sophocles/The-plays?utm_source=openai))  

悲剧的连锁终局（索福克勒斯版本）：
- 安提戈涅被判囚禁/活埋而自尽；
- 海蒙殉死；
- 欧律狄刻自尽；
- 克瑞翁在“维护秩序”的名义下把自己的家彻底毁掉，成为“政治一意孤行的空壳胜利者”。([britannica.com](https://www.britannica.com/biography/Sophocles/The-plays?utm_source=openai))
