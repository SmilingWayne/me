# 戏剧


## 戏剧分类法
-------

!!! abstract 
    <font size = 3>
    A：咱们现在干什么呢？ 

    F：我不知道。 

    A：咱们走吧。 

    F：咱们不能。 

    A：为什么？ 

    F：咱们在等待戈多。[^1]

    </font>

抓马。


- 戏剧部分，没有什么特别的偏好，看到什么感兴趣的就看。随心所欲，没有任务。包含了一些现场观看的剧目。

- 影视部分，记录一些回顾时的所思所想。多是老电影。具体影单移步豆瓣。
