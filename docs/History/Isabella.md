# 卡斯蒂利亚的伊莎贝拉


## 书中摘记

**费迪南** 
:   他们都是在即将继承王位的长兄的阴影下成长起来的，成长过程中周围都围绕着权力真空造成的战争和混乱。

**忏悔神父托尔克马达**
:   受到国王和王后的恩宠：在婚姻和精神、政治上结合。后来成为宗教裁判所的大法官。

> 9到10世纪，女性享有和男性平等的财产继承权，已婚妇女有权控制自己带来的财产，并在丈夫去世后享有其财产的10%。不过随着外来者的入侵，农民越来越依靠保护者，贵族权力逐渐增长，女性地位下降。嫁妆变得越来越重要。农民被迫接受世俗或者教会的统治，承认自己是服从他们权威的臣民。

!!! note "宗教裁判所"

    镇压对象：自称是基督徒，但是行为让其他人怀疑他们的信仰不够真诚的那些人；

    最早的一批牺牲品是**犹太人出生的基督徒**，（改宗犹太人的问题）。他们依然遵从犹太习俗，很难判断是否皈依了基督教。

    后来，同性恋者、穆斯林、新教徒等都会受到审查。

    不过，最开始宗教裁判所在西班牙收到欢迎，因为人们相信这种机构的存在是有必要的。很多人看到，伊比利亚半岛难得出现一个强有力的中央集权专制政府。

    > 每一个宗教法庭在启动的时候，都会发布“恩典敕令”，呼吁信徒前来悔罪。但是敕令的执行会导致更多人收到指控：因为想要证明自己的真诚，悔罪的人必须告发那些仍然遵循犹太教规矩的人，如果他们告发别人，就可能让别人受到恐怖的折磨，如果不告发别人，他们的悔罪就会被认为是不完整的、不真诚的。

    ------

    异端裁判所的财政方面是它迅速发展壮大的重要促进因素，因为他为政府带来了巨大的金钱。宗教法官的薪水和监禁犯人的费用来自犯人的家产。所以受到经济上的刺激，宗教裁判所毁想方设法地把嫌疑人判定有罪。

    1. 一方面，许多基督徒相信犹太人的钱是通过不正当手段获得的；
    2. 另一方面，基督徒不被允许放债，因为这是一种有害的职业，但是犹太人可以从事这种职业；
    3. 在伊莎贝拉看来，宗教裁判所是达到有价值、正当合理的目的的手段，尤其是为了战胜格拉纳达，必须得到安达鲁西亚老基督徒的支持，而这些人都非常仇视改宗犹太人。
    4. 西班牙宗教裁判所既恶毒又悲剧，从伊莎贝拉的角度看，它将西班牙团结了起来，得以消灭内部的宗教纷争。
   

对穆斯林的态度：

1. 在1492年围困格拉纳达，伊莎贝拉和费迪南承诺如果投降，穆斯林的生命和宗教信仰都将受到保护。他们也相信伊莎贝拉和费迪南会信守承诺。
2. 不到十年，伊莎贝拉就开始向穆斯林施压，勒令他们皈依基督教，违背了对穆斯林做出的承诺。
3. 大主教西斯内罗斯在格拉纳达进行了大规模焚毁伊斯兰书籍的活动，尽管他本人是个饱学之士。这是有意为之，表现了他对伊斯兰的知识和学术的明目张胆的鄙视。
4. 随着冲突和不信任进一步加剧，伊莎贝拉下令穆斯林要么受洗，要么回到北非，任何的宗教宽容都不可能存在了。

> 伊莎贝拉成功地将西班牙变成了几乎纯粹的天主教国家，但是也丧失了在西班牙生活数百和数千年的犹太人和穆斯林的勤奋和技艺。为了达成自己的目的，她在国内树敌众多，异端裁判所和其代表的宗教不宽容，成为他的遗产的一个不可磨灭的污点，在许多世代里困扰着西班牙。

## 我的人物关系整理


### 伊莎贝拉 


- 1451～1504。卡斯蒂利亚的女王。自小从天主教会处获取慰藉，受到1431年被烧死、1456年洗脱罪名的贞德的影响。

**胡安二世**
:   伊莎贝拉的父亲。1453年因为触及权贵利益被杀

**恩里克四世**
:   伊莎贝拉同父异母的兄长，国王。胡安二世去世后继位。对伊莎贝拉母女并不好，抢走了一些父亲留给伊莎贝拉的财产。是个男同。性无能，统治混乱，丧失秩序。迎娶了胡安娜。**葡萄牙公主**。

**胡安娜公主**
:   恩里克四世的女儿，由于恩里克四世与王后都生活放荡，不少人怀疑胡安娜是否是国王的孩子。

> **帕切科**，恩里克四世的权臣。1465年联合贵族控制了伊莎贝拉的弟弟阿方索，宫廷政变。迫使权力移交给阿方索。一个恋童者。在夺权后把伊莎贝拉许配给自己的弟弟佩德罗，但是佩德罗在迎接伊莎贝拉的路上**病死了**。

> 1474年帕切科和恩里克先后病死。伊莎贝拉接过权柄。没有留给胡安娜公主。1475年爆发**卡斯蒂利亚王位继承战争**：有葡萄牙血统的胡安娜是否应该继承卡斯蒂利亚王位。（阿方索五世支持胡安娜）。1479年《阿尔卡瓦条约》，伊莎贝拉获得胜利，胡安娜遁入修道院。


**阿方索**
:   伊莎贝拉的弟弟。在1465年帕切科的政变后上台，1467年病死。

> 阿方索死后，恩里克四世重新成为国王。伊莎贝拉与兄长达成了和解，但是**争取到了恩里克四世的继承人**身份。成为了王储。尽管她刚刚支持过政变的阿方索。
>
> 不过恩里克很快食言继续排挤伊莎贝拉。

**比阿特丽齐**
:   阿留瓦洛堡（也是伊莎贝拉的出生地）总督的女儿。是伊莎贝拉终身的好玩伴，也在伊莎贝拉的宫廷周旋中扮演了重要的作用。

**贡萨洛**
:   忠诚的贵族青年（贡萨洛家族），在帕切科政变扶植阿方索之后与支持阿方索的伊莎贝拉相结识。一生忠于伊莎贝拉。

**伊莎贝拉**
:   伊莎贝拉的母亲。（两牙都喜欢用母亲、祖母（父亲、祖父）的名字给自己的女儿（儿子）命名，所以出现了很多很多重名的情况）。精神出了问题，神志不清。

**罗德里格**
:   波吉亚家族的罗德里格。红衣主教、1492年成为教皇。放荡、有私生子，沾花惹草。作为阿拉贡人给伊莎贝拉提供了很多帮助。

**托莱多大主教卡里略**
:   在伊莎贝拉与费迪南结婚初期提供了很大帮助，后来因为门多萨家族得宠导致自己无法成为红衣主教，选择与伊莎贝拉做对。

!!! note "婚事"
    伊莎贝拉本来要嫁给**阿拉贡国王胡安二世**的儿子**斐迪南**，但是后来被许配给费迪南同父异母的兄弟卡洛斯，不过卡洛斯很快病死了。

    后来又要被许配给英格兰国王**爱德华四世**。但是爱德华秘密与平民女子伊丽莎白订婚导致婚事告吹（英法西之间出现了一些不和）。

    最后回到了开始，嫁给费迪南。

### 若昂一世

- 葡萄牙国王。1357～1433，1385～1433在位。


**杜阿尔特一世**
:   若昂一世的儿子。父亲去世后继位。葡萄牙国王。

**阿方索五世**
:   杜阿尔特一世的儿子，后来成为葡萄牙国王。上面伊莎贝拉的兄长恩里克四世的妻子，胡安娜，就是阿方索五世的妹妹。阿方索在位期间，领导了对非洲的征服。

> 最开始，恩里克四世计划把伊莎贝拉嫁给阿方索五世。

**布拉干萨公爵阿方索**
:   若昂一世的儿子。

**司厩长若昂**
:   若昂一世的儿子，妻子是**葡萄牙人伊莎贝拉**，布拉干萨家族人，也是**伊莎贝拉女王的外婆**。而这位伊莎贝拉的父亲是上面的布拉干萨公爵阿方索。也就是说她嫁给了自己的叔叔。（日常了）

**“航海家”恩里克**
:   若昂一世的第三子。航海家。
