# 苏联：铁幕🪖、核弹🚀、柴可夫斯基🎻

!!! Tip "咳咳。"
    重写首记于2024.04。可能苏联，或者说俄国，是最有反差感的一个。铁幕、核弹、柴可夫斯基，乌拉尔山到西伯利亚，这块地太神秘了。有一种在古拉格演奏交响乐的感觉，定睛一看是在冻土上搓核动力圣地（老文明6了）。**20世纪的苏联史，我尤其喜欢把所有相关人物的生卒年都写上，因为...你看到生卒年，就知道了所有。**

    苏联史，本科时候不好好学习，泡图书馆看一些有的没得，启蒙是沈志华老师的苏联史。俄国应该是几个主要文明里我最先读完近现代史的。当时应该是2021年，我思想最左的一年。当时不是很会记录笔记，花了很长一段时间在整理上，现在回过头来看，一些内容觉得幼稚，毫不犹豫地删掉了，一些内容依然留着。

    现在，按照当时我反复阅读并且记录的内容，修订整理成新的文本，给那段充满了飞雪和鲜血的历史一个敬畏的凝视吧。

## 一、大清洗的权力机器

!!! note "大清洗的核心执行者"

    **维辛斯基（1883-1954）**——苏共最高法院军事法庭总检察长，本为孟什维克，1920 年加入布尔什维克。吊诡的是，一个原本的孟什维克却把大量老布尔什维克送上革命的断头台。1935 年担任总检察长，在大清洗中提供理论基石：**"刑法是阶级斗争的工具，口供是证据之王"**。他自称：==我从不相信抽象的正义。== 为获得斯大林赏识，甘愿卖命，是少有的得以在大清洗中保全的高层人物，战后负责苏联外交关系。

=== "亚戈达 (1891-1938)"

    亨利希·格利戈里耶维奇·亚戈达，苏联国家安全部门首脑，联共（布）中央候补委员、委员，国家安全总政委，一级集团军级政委。**死于"肃反不力"和保护一部分受害者**——他本人成为自己参与建造的机器的牺牲品。

=== "叶若夫 (1895-1940)"

    尼古拉·伊万诺维奇·叶若夫，苏联内务人民委员部首脑，斯大林肃反运动计划的主要执行者。学历不高，行事心狠手辣，在大清洗转向时被清算。1938 年 1 月党中央作出《关于错误地解雇因反革命罪被捕者的家属这一事实的决定》，呼吁"不要无充分理由就控告人"。1938 年 8 月贝利亚成为他的助手，11 月 25 日他被迫辞职，1940 年 2 月被处决。

!!! quote "安东诺夫暴动 (1920-1921)"

    坦波夫叛乱，苏联官方称安东诺夫叛乱，是俄国内战时期布尔什维克统治下**规模最大、组织最严密的农民叛乱**。发生地在莫斯科东南不到 500 公里，具有社会革命党（左翼）背景，图哈切夫斯基负责镇压。**用毒气解决了战斗**。乌赫塔集中营成为苏联集中营制度的开端。

## 二、托洛茨基与反对派理论

!!! example "社会主义原始积累理论"

    **普列奥布拉任斯基**作为左倾经济学思想家，提出"社会主义原始积累"理论，主张**牺牲农民利益获取工业化所需资金**，这一理论在斯大林模式建设初期被应用，但苏联一直未能完成纠偏运动。

    核心观点：

    - 实现手段：社会主义国家运用预算、信贷、货币价格等手段，重新分配国民收入，**把资金从农业抽调到工业**
    - 根本目的：尽快在落后的农业国实现工业化
    - 托派认为：**苏联生产力太落后，难以凭自身力量快速工业化**；资本主义国家敌视封锁，苏联不能获得先进技术；苏联对世界具有依赖性，不能脱离国际分工

    结论：**社会主义不可能在一国之内建成**。解决方案：停止新经济政策，通过"剪刀差"从农业抽取积累，集中发展大工业，强调高速度。

!!! note "不断革命论与中国"

    托派的不断革命论**没有阶段性划分**。十月革命阶段在俄国可行，但对中国革命不正确——国情太特殊，不允许冒进，没有俄国那样的社会基础和革命动能。托派反对国共合作，认为**一旦革命停止，修正主义分子就会出现，资产阶级会篡夺革命成果**。

    横向看，不断革命论强调**世界范围内的共产主义革命**。托洛茨基强调革命对于一国民众的自发性，认为"没有严密的工人组织不可能强加一个来自于他国的革命"，本国共产党对马克思主义的创造性运用是革命成功的关键，他国支持只是推波助澜。

!!! quote "对原始积累论的评价"

    > 苏联靠**内部放血**迅速实现工业化，但没有给农民足够的支持和购粮款，某种意义上合法化了"社会主义原始积累"，从农民身上抽取了资金。
    >
    > - 强迫国内"节约"
    > - 绝对命令式的"赶超"计划
    > - 谁指出了轻工业的不足，谁就会被定义成"右倾"

!!! note "斯大林击败托派后的现实"

    斯大林击败托洛茨基后，**在经济上借托派的社会主义原始积累理论搞定了工业化基金**，通过和资本主义国家做生意，以农产品换工业设备取得外部支持，利用甚至牺牲国际共运的一部分利益，为苏联建设创造有利外部环境。

    > "斯大林模式下的苏联，更像是梵蒂冈，领袖就是教皇，克里姆林宫就是圣伯多禄大教堂。"

??? question "我的思考"

    一个严重的问题：**在对生产力低下、经济落后地区进行建设时，如何设定目标？**

    1.  一个反驳点：先让他们吃饱饭再说；
    2.  一个矛盾点：如果不愿承认与外国的巨大差距，人们反而安于刚脱离贫困的现状。很多地方有突破贫困的能力，但人们往往选择安于贫困，对外界一无所知，助长了官僚主义气焰。托派看到：很多地方官员吹嘘经济建设成果，反而束缚了农村发展活力。

## 三、列宁遗嘱与权力继承危机

!!! note "列宁遗嘱的政治化"

    列宁遗嘱对托洛茨基的评价："**过分地热爱行政**"；对斯大林的评价："**无法理智地使用权力**"。

    遗嘱成为**党内政治斗争的工具**：

    - 托派用列宁遗嘱攻击斯大林："无条件承认列宁在遗嘱中..."
    - 遗嘱的公布与否成为博弈焦点
    - 总书记的权力问题被刻意回避
    - 中国革命的路径问题（对共产党的态度）
    - "一国建成社会主义"：如何对待外国？是敌对宣扬特殊性，还是不脱离国际争取支持？
    - 共产国际的领导问题：必须强迫别国共产党接受苏联方针吗？

    《我们怎样改组工农检察院》被政治局篡改。托派与当权派的分歧逐渐从策略性转向纲领性。

=== "三驾马车"

    季诺维也夫、斯大林、加米涅夫组成"老近卫军派"核心。七人小组违背中央意志，可任由一人提议增加或取消政治局会议上提出的问题，实际控制了权力范围。

=== "在册权贵"

    "在册权贵"花名册成为任命制的工具，省委书记成为可轻易任命的职位，中央监察委员会职能被削弱。

## 四、新经济政策的辩论与终结

!!! note "退却还是进攻？"

    - 退到国家资本主义，不退了，这就是最后的界限
    - 党内希望一旦好转，就转向进攻
    - 合作社：控制"小生产"，过渡到"大生产"

| 时间      | 事件                                                                 |
| --------- | -------------------------------------------------------------------- |
| 1921.2.16 | 俄共十大，贸易自由被视为反革命口号                                   |
| 1921      | 饥荒歉收，人口大量减少；胡佛总统派救济署提供近 80 万吨粮食至 1923 年 |
| 1922      | 列宁对小生产和农民问题不放心；外贸垄断问题；没收教会珍宝赈济灾民     |
| 1923      | 列宁认识到必须团结农民、取信于农民；病情加重，逐渐摆脱具体事务       |
| 1923.3    | 列宁脱离政治生活（"他是为政治而生的"）                               |
| 1924.1.20 | 列宁逝世                                                             |

!!! quote "宗教政策与世俗国家建设"

    取消神学必修制度，东正教持红白中立立场。1922 年没收教会赈济灾民，列宁指示：**对任何抵抗可以坚决镇压**，枪决黑帮分子，本质是打压宗教势力建立世俗国家，获取灾区贫民支持，削弱教会争取群众。1928 年东正教承认苏联世俗国家属性，1929 年同宗教组织宣战，拆除教堂修道院，建立监狱疗养院，宪法将宗教限定在教堂院内。

=== "农业合作社"

    互利原则下的城乡大规模商品交换：采购机器、生产工具、加工初级原材料。1922 年土地法典出台（土地所有者拥有使用、占有和支配权），农民可退出土地社选择土地利用形式。村民互助社调动群众积极性，被贫困群众支持。

=== "消费合作社"

    组织商品交换，农业信用社、共耕社、产销协作社共同发展，提供足够农具和生产资料，推动工业化、自动化、电气化。

=== "布哈林的立场"

    布哈林利用税收杠杆调整富农，不建议加大打击力度伤害中农，主张工农结合：

    - "一个农业国却输出工业进口粮食"：必须先提高农业水平，个体经济尚有刺激空间，不应与集体农庄对立
    - 应对资本主义进行交涉，争取生产力支持
    - "向农业让步"
    - 李可夫称之为"政治上的近视"
    - 斯大林指责为"让步主义"

    1928 年布哈林秘密会见加米涅夫（托派），认为斯大林"危险"，此事日后成为托派反击和布哈林被攻击的把柄。《一个经济学家的札记》（1928 年载于《真理报》）。

!!! note "粮食收购危机 (1927-1928)"

    斯大林秘密视察西伯利亚（任内最后一次），认为是富农私藏粮食、哄抬物价、投机倒把，严厉打击持同情心态的党员，加剧党和农村割裂。强行征收粮食，有无余粮成为是否逮捕的标准。

    **后果**：失去中农支持，激化贫农矛盾，人们认为"为自己播种反而减少了播种数"，一些贫农仅因持有种子就被列入投机者行列。粮食收购机关权力膨胀。

    李可夫警告、布哈林担忧："过分强调工业产品输出，国内商品荒废，中农、富农、一部分贫农都在反对苏共"。


!!! quote "新反对派的形成"

    1925 年新经济政策推广至农村，布哈林不惧怕农村资本主义滋生，季诺维也夫和加米涅夫为首的"新反对派"担心农村资本主义发展，坚持阶级斗争立场。布哈林"发财吧"口号被新反对派打成"帮富农说话"。斯大林为打击新反对派而为布哈林说话，采取既批判又辩护的态度。

    **核心矛盾**：小经济农村与国家垄断工业之间的矛盾。

## 五、工业化与集体化

!!! note "工业化的资金来源之争"

    | 观点         | 主张                                                         | 代表人物 |
    | ------------ | ------------------------------------------------------------ | -------- |
    | **外债论**   | 取得贷款、恢复生产、农业、交通，必须善于利用资本主义利益矛盾 | 克拉辛   |
    | **自力更生** | 驳斥举借外债，认为外债必然伴随奴役性和剥削性                 | 斯大林   |
    | **内部积累** | 靠本国节约完成原始积累，国内市场需求和资源是发展基础         | 主流派   |

    轻工业负荷不足，重工业对日用品有大量需求。输出农产品进口设备的政策面临资本主义包围，如何发展对外贸易成为关键。

!!! quote "商品荒与治疗"

    在单纯与市场脱离的政策下，单纯提高生产来医治商品荒是没有前途的。**剪刀差**导致农民买工业品需付钱，卖农产品时少付钱，农业集体化成为必然。

=== "沙赫特案件 (1928)"

    逮捕矿井专家，一些没有物证只有口供的定罪："资产阶级专家策动的反革命"。实际上因设备原因忽视安全，官僚主义泛滥（11/53 被处决）。**转嫁矛盾**，通过"破坏者"化解危机，借助对知识分子不信任炮制舆论，将阶级敌人破坏作为经济困难根本原因。

=== "工业化成就与问题"

    1932 年斯大林宣布苏联完成工业化：

    - 片面强调产值和产量，对路货物、优质货物奇缺，品质单调
    - 行政命令和严酷惩罚机制
    - 冒险主义速度导致严重问题：只顾工业成果，不顾经济核算
    - 1933 年允许农庄成员在市场交易买卖
    - 第二个五年计划："彻底消灭资本主义成分，彻底消灭产生阶级差别的资本主义残余"
    - 轻工业仍未达目标，兴建机械厂、工厂、军工工厂

    > 枪毙 600 人，给 2000 人糖果吃。
    >
    > 让劳改犯们去开拓北极圈吧！地质、矿产、建筑学家被流放，对劳工军事化管理。

!!! note "农民的国家化"

    - 国有拖拉机站控制
    - 生产计划国家制定
    - 对自留地中牲畜严格限制
    - 强制将农民拉入"集体农庄"
    - 谷物价值价格为成本的 15%，打击社员积极性，引发抵制怠工

    1929 年全面批判布哈林、李可夫。1929.12.21《真理报》（原被布哈林控制）给斯大林献忠。农业全盘集体化、完全消灭私有农民。政策不符合常理地大规模转变：百姓不理解，官员不明白、条件不允许。设立过高指标、不可行要求、脱离实际。

    中央宣称"胜利"，实际上 6% 的加入者退出集体农庄。

## 六、党内斗争的演变

!!! note "从工人民主制到任命制"

    党的十大和"工人民主制"，两派都提出"官僚主义对国家和政党机关的压力"：

    - **工人民主制**：主张工人地位上升，参与政治
    - **机械的集中制**：必然由派别活动补充，需听取群众声音

    自那以后，会议争论往往成了对托洛茨基的人身和经历攻击，成为人格化矮化党员的会议，"扣帽子"盛行。党内组织涣散、群龙无首，不利于党内政治生活建设。

!!! quote "苏共十大会议第一条"

    **常委可以不经过讨论，不受监察委监察情况下开除委员出中央委员会。**

=== "斗争的扩大化"

    - 对于对手的副手、助手、党羽，虽无直接权力影响但有影响力的人士逐渐剥离组织权力中心
    - 解除对手的眼线和信息来源，分化对手权力圈
    - 实际上剥削了中央监察委员会的职能
    - 1926 年斗争尖锐化：反对派让步，当权派得寸进尺，使用脏手段，"秘密集会"出现

=== "13 人政纲的分歧"

    | 议题           | 反对派立场 | 当权派立场 |
    | -------------- | ---------- | ---------- |
    | 资本主义稳定性 | 不稳定     | 相对稳定   |
    | 一国社会主义   | 不可能     | 可能       |
    | 官僚主义       | 尖锐化     | 可控       |
    | 热月化         | 正在进行   | 否认       |
    | 超工业化       | 支持       | 谨慎       |

=== "派别主义的禁止"

    1927.11.14 开除出境；1927.12 苏共十五大通过五年计划；1927.12.18 对"反对派"决定：开除反对派在中央委员会中的职务。

    策略性分析转向纲领性分歧：

    - 不断革命：社会主义革命的国际属性、世界革命论
    - 无产阶级专政
    - 什么是社会主义：如何确定社会主义标准
    - 农民和工业问题：剪刀差？
    - 托派寄希望于"先在西方资产阶级国家无产阶级革命胜利，用最先进技术帮助俄国提升生产力"

!!! note "橡皮图章"

    橡皮图章（Rubber stamp）是政治比喻，指代理某些法理上权力很大而实际权力很小的人或机构，只执行职权，没有主要权力但必须设立以维持程序合法性。橡皮图章几乎不会否决或不能否决被要求批准的提案。

    > 专政政体的立法机关便是常见的橡皮图章，只会一味通过法令。立法过程更多是为了营造表面合法性与专政和谐统一的假象，而非因为该机关具有相对应权力。

## 七、文化与意识形态控制

!!! note "哲学船事件 (1922)"

    哲学船是被苏联驱逐的一批知识分子的统称。1922 年 5 月，列宁致信契卡领袖捷尔任斯基，首次提出将支持反革命的知识分子驱逐出境。6 月捷尔任斯基提交针对"反苏团体"报告，政治局决定成立委员会负责拟定被驱逐名单。8 月起抓捕相关人士审讯、判决。共 100 多位知识分子乘两艘德国船只"哈肯船长号"与"普鲁士号"从彼得格勒前往德国斯德丁。包括尼古拉·别尔嘉耶夫、尼古拉·洛斯基、谢尔盖·布尔加科夫等。

=== "文化专制体制"

    - 作家协会成为满足党无产阶级文化需求的机构
    - 布哈林：无产阶级专政的长期性
    - 列宁：不需要共产主义文化，有资本主义文化也够用，主要教工人识字、卫生知识
    - "在文学领域内夺取阵地"：反对投降，反对摆共产党员架子
    - 党用一切办法反对对文学事业的专横和行政干涉
    - 反对在艺术作品中百分百思想纯洁性

=== "哲学与社会科学批判"

    - "全方位打击"，对象包括恩格斯、黑格尔等人
    - 德波林学派被批判，波及自然科学领域学者
    - 采用"军事性质"的书报检查制度
    - 学术问题上升到政治问题："反机械论"
    - 一些问题最终以斯大林给出的"政治结论"为终点

=== "扫盲运动 (1927-1939)"

    课本、教师培训、兴建学校。到 1939 年识字率显著上升，除文盲运动取得丰富成果，为少数民族地区规范文字。对应 50-60 年代在华学校调整，"夜校"。**为工业化服务的教育**。

    但高度集中的文化体制下，互相检举、批判成为常态，迎合、颠倒黑白成为政治正确。


!!! quote "书报检查制度"

    政治书报检察官拥有文字批评职能，一个同时发挥**宗教审判**与**最高文艺法官**功能的职位。大部分书报检察官文化水平不高，却可对可能流露细节情绪的地方作修改。"掂量字里行间的小情绪"。


!!! note "日丹诺夫主义"

    反对世界主义，一种文化专制。"战前大批判"。日丹诺夫试着建构新艺术哲学，主张大量减少该域文化，改用简单、科学性的图表或符号象征某些道德价值。


!!! quote "路标转换派"

    关闭学术期刊：在经济落后同时，拧紧政治螺丝钉。禁止苏俄机构聘用被驱逐的人为代表。既认为"可以凭借自己的努力为共产主义建设做出贡献"，又认为"必须限制知识分子"。对待不同信仰学者戴上"反苏维埃"帽子。

    1917 年立宪会议，农民投票给社会革命党人，布党政权掌握不牢靠，驱散立宪会议。认为"无产阶级处于俄国少数，不能让农民与无产阶级享受同样选举权"。农民不满，抵制选举，喊出"建立农民政党"口号，被视为威胁苏维埃统治基础，大量剥夺农民选举权，强行划分部分农民为富农。

## 八、后斯大林时代的改革尝试

!!! note "赫鲁晓夫的改革"

    赫鲁晓夫（1894-1971）：煤场经理出身，借助政治斗争上位，是大肃反帮凶，9979 名党员被开除。投其所好骗取上层信任。

    - 1938-1949 在乌克兰工作：工业农业上升，短暂远离莫斯科
    - 1949 回到莫斯科，掌握内部政治斗争之道
    - 只忠于首领，不忠于人民，只为特权，不为国办事

    改革措施：

    - 大量垦荒、种玉米、效仿美国；粗放广种薄收与精耕细作矛盾
    - 改组拖拉机站、收割机站，让机器和农民联合，但操之过急，耗尽农庄财力，零件配给跟不上
    - 扩大集体农庄中农户个数
    - 从国家管理到党的管理
    - 核军备竞赛（军事科研占总科研 80%），习惯于"宏大计划"


> **"马林科夫与利别尔曼建议"** :马林科夫（部长会议主席）主张提高人民消费品生产量，加速轻工业发展：
> 
> - 工业企业权力下放
> - 进化到改组党组织：取消党委会，专设工业党委会
> 
> 利别尔曼建议：**用经济手段代替行政手段影响生产**
> 
> - 只有两个指标：数量、交货日
> - 生产率、工作日、工资、新技术由企业自行制定
> 
> 莫洛托夫、卡冈诺维奇、马林科夫形成反党集团。

!!! note "赫鲁晓夫的秘密报告"

    只反斯大林，不反体制。起诉和审判不经过法庭，没有按党章就处决二号人物，少数人密谋可杀死二号人物。

    - "医生谋杀案"：捏造的，认为医生故意缩短斯大林生命
    - "明格列尔人案件"的错误定性
    - 纠正民族政策问题
    - 纠正军费开支在国民经济上的负面影响
    - 扩大集体农庄中农户个数

=== "戈尔巴乔夫改革"

    | 领域     | 措施                                             |
    | -------- | ------------------------------------------------ |
    | 企业     | 给予更多自治权                                   |
    | 理念     | 更加关注"民主"和"人道"                           |
    | 历史     | 拨乱反正                                         |
    | 经济     | 主张直接进入市场经济阶段                         |
    | 农业     | 取消计划，取消指令性指标，承包制改革             |
    | 价格     | 对长期形成的价格体制改革（价格监督机制）         |
    | 金融     | 财政信贷改革、银行信贷体制                       |
    | 对外     | 对外经济体制改革、"理智妥协"                     |
    | 意识形态 | 允许不同意识形态存在，扩展国际空间，争取国内改革 |

    > 《改革与新思维》：对内允许不同意识形态存在，目的是扩展国际空间，争取国内改革。

!!! note "改革失败的原因"

    1.  企业领导人压力大：风险大、责任大
    2.  人们不清楚已开始改革的实质和意义，未及时享受成果
    3.  改革没有从农业开始
    4.  中央控制不了地方、法律失败
    5.  思想混乱、否定过度，降低党的威信
    6.  斯大林模式的制度性缺陷、矛盾积累和激化

    东欧政策主张各国经济互利和政治斗争。施行多党制实际上是政治体制改革不断妥协的最坏结果。由于苏联前期施行大国沙文主义，各加盟共和国对斯大林体制十分不满。

!!! quote "休克疗法与叶利钦时代"

    叶利钦的休克疗法、快速融入西方动机，强调经济改革的迅速、全面，国有化转私有化，限制政府作用。

    **后果**：

    - 全面放开价格后，超级通货膨胀出现
    - 大量企业私有化未导致效率提升或盈利，而是催生大量寡头
    - 企业内部管理层为确保控制公司形成既得利益集团
    - 人口减少，官僚政治、休克疗法从根本上破坏苏联工业和科学技术基础
    - 大量科技工作者移居国外，企业不愿大规模投资
    - 寡头官僚相互勾结控制经济，俄罗斯进入资本主义世界体系逻辑，成为西方原料供应地
    - 代议制政府和总统制对抗：互相攻击（1993 年）
    - 叶利钦联合寡头促进选举成功，官僚与寡头（金融家、石油巨鳄）联合执政

    - 格鲁吉亚问题尖锐化：给俄共成员党内处分与警告

    - 沙皇的侵略政策：北方四岛、波兰、高加索山以南、格鲁吉亚
