# 东风：法国知识分子与60年代的遗产

> **"马克思与可口可乐的孩子"——六八一代的遗产与法国思想史的转折**

---

## 引言：东风的隐喻

!!! quote "作者的反思"

    **我呼唤东风，仔细说来，我呼唤那东风里的真理，我呼唤那东风告诉我的真相。**

    这本书应该和五月风暴、巴黎学运、加缪萨特之争、存在主义哲学结合起来看。

    一阵东风从大兴安岭刮到卢瓦尔河，这阵东风早已不是它本来的样子了。所谓的"革命理想"并不是复刻，而是西方主义的修正。巴黎人懂不了东方的种田农民，因为巴黎人不需要种田，他们也不需要管理这么大的国家，他们也没有经历过我们这样的近代史。

    但是，这至少是一阵东风。**东风中有真理，就像亚当斯密、李嘉图的理论中也有真理。真理不会因为它诞生在东方或者西方就丢失它原来的光芒。**

    总有一些事情，放在当时那个时代，是无比正确的。**虽然在周围人眼里都差劲极了。**

---

!!! note "知识分子的本质：介入"

    **知识分子之所以存在，就是因为这个群体具有意志引领、改变现实的潜在动力。**

    - "介入"的思想是理解法国知识分子传统的钥匙
    - 脱离了实践，脱离了"去粗取精"的思路，知识分子不过是在闭门造车而已
    - 十八世纪以来，法国知识分子以"世俗贵族"自居——**正义 - 真理与美德的仲裁者**

    可以看作半部法国思想史。了解法国之后，反而是对他的数学、哲学更加感兴趣了。

## 第一章：六八一代的时代精神

!!! abstract "马克思与可口可乐的孩子"

    **如果当前的历史能够根据弹道导弹、热核武器、月球竞赛与军备竞赛定义的话，那一部分学生徘徊于嬉皮士的否定、救赎性暴力的渴望与向新乌托邦的逃逸之间。**

    学生对旧制度教学制度的反感：**"贩卖知识的机器"**

    ==情境主义国际的批判==

    - **我们不想生活在一个以"死于无聊"为代价来确保"不死于饥饿"的国度中**
    - 宫廷制度的残喘：法国行政制度的残留，社会精英和仆从间的鸿沟
    - 法国组织层仍然是等级制的，上层垄断、跃升渠道变窄

    ==日常生活的革命==

    消费主义、诱惑改变了传统的日常生活。兰波的呼唤：**"Changez la vie~"**（改变生活）

---

!!! info "国际政治背景：1960s 的关键事件"

    | 事件                     | 时间      | 影响                                               |
    | ------------------------ | --------- | -------------------------------------------------- |
    | **布拉格之春**           | 1968      | "具有人性面孔的社会主义"被碾碎在苏联坦克的履带之下 |
    | **越南战争**             | 1955-1975 | 法国接管奠边府失利后的法属支那，成为左派关注焦点   |

    > 对于这些巴黎高师的左翼学生，对中国了解越少越有利。对于激进政治思想来说，**"文革"成为投影屏幕，一项"罗夏测试"**。
    >
    > 上/中层阶级精英的孩子们——**一个左派政治想象的投射：议题已经变成了"我们脑中的中国"**

## 第二章：法国毛主义与激进政治

!!! note "亲华妄想与文化大革命专号"

    **巴黎人将"文革"作为左派政治想象的投射——那是"我们脑中的中国"，而非真实的中国。**

    - 戈达尔的《中国姑娘（La Chinoise）》放大了巴黎高师学生的亲华妄想
    - 索莱尔斯《太凯尔》杂志推出"文化大革命专号"
    - 这些先锋学生不少成为了"法国文化与政治人物"，进行**"一场体制内的长征"**

    风波后，人们更进步地把"列宁主义"丢进了历史的垃圾桶。

---

!!! quote "福柯与权力的微观物理学"

    **福柯：文森大学，"鉴于信息小组"，支持狱中激进毛主义的团体。**

    ==核心口号==

    - **"革命万岁小组！Viva La Revolution~!"**
    - **"到群众中去！"** | **"下马观花"**

    ==权力的微观物理学 (microphysics of power)==

    福柯的思想从宏观权力结构转向微观层面——监狱、医院、学校中的权力运作机制。这一转向与他对毛主义"到群众中去"的理解相互呼应。

    > 蓬皮杜逮捕毛主义者，销毁出版物，只会让他们更多地成为殉道者。

---

!!! warning "暴力对抗学说：布鲁亚事件与阶级对立"

    **1972 年，工人阶级女孩布鲁亚在安纳特瓦被虐杀——工人阶级的愤怒：认为是资产阶级对工人阶级的一种杀戮。**

    ==萨特的立场==

    二战后，工人阶级被白领、工薪阶层所取代，但萨特和他的学生致力于放大工、资对立。

    - **"走群众路线" (mass line)**
    - **"先红后专" (First red, then expert)**
    - **"一个，两个，无数个越南！"**
    - 萨特认为需要左派自己的**"人民革命法庭"**

    ==暴力对抗学说的核心==

    > "暴力对抗学说"
    > "共产主义教义问答"
    > "他们正在埋葬的是左倾主义"
    > **"受辱的民众可以采取短期的恐怖行动并攻击一小撮卑劣的个体"**

    现代西方学界多放大五月风暴中"反专制主义的精神"，认为它"滋养了极权信条"。

## 第三章：思想论争与分化

!!! abstract "萨特 vs 加缪：政治暴力与伦理真空"

    | 萨特                           | 加缪                     |
    | ------------------------------ | ------------------------ |
    | 政治暴力！反抗欧洲人的救赎性质 | 陷入伦理和政治的真空地带 |
    | 主张人民革命法庭               | 拒绝暴力正当化           |
    | 介入现实，改变世界             | 保持道德纯洁性           |

    **存在主义与结构主义者的争论**构成了这一时期法国思想史的主线。

---

!!! note "毛主义者的分化：什么是正统？"

    **五月起义后，毛主义者面临根本问题："什么是正统？如何做到实事求是？"**

    <span style="color:red;font-weight:bold">两大派别</span>

    1. **教条的、列宁主义的正统派**
       - 坚持传统的革命路线
       - 强调组织纪律与先锋队理论

    2. **聚焦日常生活政治的派别**
       - 妇女解放运动
       - 同性恋权利
       - 生活方式的政治化
       - **"小资产阶级个人主义" (Petty Bourgeois individualism)**

    <span style="color:red;font-weight:bold">对法共的批判</span>

    1964 年，法国共产主义运动联盟脱离了法共，并指责法共**"不充分的斯大林主义"**。

    > 你们都是**"意识形态教条主义者！是意识形态教条主义的囚徒！"**

---

!!! info "法国知识分子群像"

    **十八世纪以来法国知识分子以"世俗贵族"自居——正义 - 真理与美德的仲裁者。**

    **结构主义与后结构主义思想家谱系**

    | 思想家              | 核心贡献                       |
    | ------------------- | ------------------------------ |
    | **罗兰·巴特**       | 符号学、神话学、作者之死       |
    | **德勒兹**          | 差异与重复、欲望机器           |
    | **格里耶·布罗代尔** | 长时段历史、物质文明           |
    | **拉康**            | 精神分析的结构主义重构         |
    | **福柯**            | 权力 - 知识、规训与惩罚        |
    | **鲍德里亚**        | 消费社会与广告、营销、拟像理论 |

    > 消费社会、平庸化、异化的中心在于：**流通领域，在于符号学。"消费者选择"**

## 第四章：五月革命的遗产

!!! quote "直接民主与工人管理"

    **五月革命的标志：直接民主与工人管理的思想从外围转移到中心。**

    左派政治想象的转变:

    - 议题从真实的阶级斗争转向**"我们脑中的中国"**

    - 戴高乐：**"谁会真相信我 67 岁才开始独裁者的新事业呢？"**

    圣茹斯特的警告:

    > **"人不能进行纯洁的统治！"**
