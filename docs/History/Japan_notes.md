## 古坟时代

大和（Yamato）政权的出现。出现了很多大方圆墓。地区贵族联合体。氏族集团的权力膨胀。

> 大仙古坟：最大的坟墓。可能是埋葬某个天皇。

        - 氏民
    
    - 部民制：划分生产阶级的制度
    
    - 县主国造制：地方贵族担任的“县”的首领，氏姓制度的补充；
  
- 大和政权与半岛展开贸易和战争：维持影响力（新罗、百济、高句丽）。562年，失去了在朝鲜半岛的领土，中华大陆也无暇顾及。

- 539年，苏我、物部借天皇继承问题攻击大伴金村，使其被罢免。

## 飞鸟时代 

- 593年，圣德太子推举为摄政王。很得民心。
- 594年，圣德太子影响下，佛教成为日本国教。
    - 开始打破过去的氏族制度。
- 631年:17条宪法。
- 607年：小野妹子（男）出使隋朝。裴世清随小野妹子回访。

后来，土地兼并加剧。推古天皇和圣德太子去世后，苏我家族更加猖獗，专横跋扈。苏我虾夷、苏我入鹿，将圣德太子的儿子逼迫致死。

### 大化改新

改革派：中大兄皇子（38th，天智天皇）+中臣镰足。

645年，斩杀苏我入鹿，迫使苏我虾夷自焚。拥立孝德天皇即位，《改新之诏》。从奴隶社会过渡到封建社会，建立完备的律令制度、天皇为首的中央集权制度。

中臣镰足因其家族的重要作用，赐姓藤原。

依照隋唐经济政治制度进行改革：取消部民制度、实行公地公民制度，收归国有、班田收授法，编制户籍。

但是，中央为了争取到一些贵族的国防支持，给一些氏族贵族保留了私有田、私有民：为后续埋下了隐患。

!!! note "租庸调傜役制"
    - 农民缴纳田租
    - 地方必须提供一些特产进贡中央
    - 义务兵役制与农民强制服役

!!! note "两官八省制"
    仿照三省六部制度设置。

## 奈良时代（710-794）

唐风的平城京。日本的疆域开拓。北方的虾夷和九州的隼人，都被纳入统治。

!!! note "虾夷人"
    和阿依努人应该是同一组成。



### 藤原氏的崛起

氏族集团阻断阶级流动，氏族地位不断提高。镰足之子，藤原不比等担任太政大臣，不比等还把自己两个女儿嫁给先后两个天皇。

740，藤原广嗣起兵反抗遣唐回来的橘诸兄等新贵族阶层。

藤原仲麻吕（不比等之孙）借助姑母势力迫使橘诸兄隐退。仲麻吕最终当上太政大臣。废除孝谦天皇，拥立淳仁天皇。二者矛盾激化，仲麻吕讨伐孝谦失败，淳仁复位。


## 平安时代（794-1185?）

平城京-> 平安京。私有制庄园、“摄关政治”、国风的形成。


!!! note "背景"
    班田制的瓦解（田数减少、行政能力低下）与大庄园的出现。

    农民的反抗、寄身大贵族名下逃离管辖，大贵族开始圈地，形成更加大的私地。

藤原冬嗣：“藏人头”的头衔。将女儿嫁给天皇，开启“外戚干政”。 

59th宇多天皇，“万机巨细，百官总已，皆**关白**于太政大臣，然后奏下”。

!!! note "摄政+关白 => 摄关政治"
    摄政，天皇幼年时期的辅佐统治者；
    关白，本来是禀报，天皇成年后的辅佐者。
    
> 庄园主争取“不输不入”的特权，不向国家进贡，也不从国家拿东西。完成了土地私有化的进程。

藤原氏是最大的庄园持有者，掌握了雄厚的经济地位，同时积极培养女性，通过皇亲国戚实现干政。

藤原北家的藤原道长时期（966～1028），摄关政治达到鼎盛。曾说：“此世即我氏，如满月无缺”。

!!! note "源氏物语"

    贵族社会由盛转衰的时代，紫式部也是藤原氏。多用假名进行书写，而不像是男性用汉文进行书写。

平假名：9世纪开始使用；

片假名：10世纪开始使用。

到藤原道长之子藤原赖通将女儿宽子嫁给天皇后并没有生子，导致藤原对朝政的控制力减弱。1068年在位的后三条天皇是170年来唯一不是藤原氏所生的天皇，联合其他贵族迫使赖通辞职，摄关政治开始衰落。

!!! note "院政"
    “退隐的政治”，皇权抵抗摄关政治发展的政治制度。

    1069年，天皇开始将庄园收归国有。

    天皇让位，自己当“上皇”，执掌朝政。 命令源氏和平氏武士团作为主要成员担任守卫和执行工作，加强了武士的地位。

!!! note "武士的起源"

    庄园时期，庄园主为反抗政府、兼并其他庄园，武装自己的庄民，形成一些职业武士。包括一些神社、寺院也组织自己的武士部队。

!!! note "平氏｜源氏"
    939年，**平将门之乱**。

    桓武平氏、伊势平氏

    “承平天庆之乱”

    1031年，后来源氏因为平平氏叛乱有功，开始称霸关东地区。

    1091年，朝廷遏制源氏，源叛而平起。

    **平清盛**（1118～1181）通过“保元-平治之乱合战”让平氏家族达到鼎盛：
    
    > 后白河天皇、崇德上皇、源赖朝（流放到伊豆半岛） 

    藤原氏迅速衰落，武士阶层成为日本社会主要力量。

    - **源平合战（治承·寿永之乱）**

    1180年，源赖朝以镰仓为基地，击败了讨伐的平氏。
    
    1181，平清盛去世，平氏迅速衰退。
    
    1184年，武将源义仲将平氏赶出京都。后被源赖朝杀。

    源赖朝在 **一之谷之战** 击败平氏，试图将平氏赶尽杀绝。

    1185年，在长门国外的海上，坛之浦之战。平氏族战败。家族成员和安德天皇沉入大海。天皇所拥有的三件神器：八尺镜、八尺琼勾玉、草剃剑也随之沉海。

后白河天皇试图挑拨源氏矛盾未果，1189年，源赖朝征服了奥羽地区，1192年，获得天皇所赠“征夷大将军”称号。遂为第一任大将军。建立镰仓幕府。开启镰仓时代。由此产生了武士政权，武士与贵族的对立开始走上舞台。

## 镰仓时代（1185～1333）

- **御家人制度**、“侍所”机构，“别当”、各地“守护”担任各地统治者，武人开始组建与京都并立的二元政权。

- 公家政权和武家政权的对立

!!! note "源氏内部的不稳定"
    源赖家排挤幕府元老，在北条家（北条政子是源赖朝的妻子）的策动下遭到罢黜，北条家建立元老协议制。毕竟北条时政曾经帮助源赖朝建立幕府。

    通过政变，北条时政分割将军权力。到时政之子北条义时担任“执权”时，1121年。皇族策动不满幕府的贵族以及武士造反“承久之乱”，但被幕府镇压。

    武家开始专权。

!!! "元日之战"
    1274 在北九州，元朝和高丽的联军与御家人交战。“文永之役” （对马岛之魂）。失败。

    1275年，北条时宗斩杀了元朝使节，拒绝遣使。

    1281年，“弘安之役”，“神风”。

    通过两次战争，北条家族强化了自己的统治。
