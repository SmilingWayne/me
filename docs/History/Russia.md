# 俄国史

俄罗斯千年变革的政治、领土与文化年鉴。

## 总览

## 🔍 四个概念

| 概念             | 类型            | 时间范围    | 核心特征                                                              |
| ---------------- | --------------- | ----------- | --------------------------------------------------------------------- |
| **基辅罗斯**     | 🏛️ 国家/政权名称 | 882–1240年  | 东斯拉夫第一个统一国家，首都基辅，基督教化起点                        |
| **莫斯科公国**   | 🏛️ 国家/政权名称 | 1283–1547年 | 蒙古统治下的地方公国，<br>后成为统一俄罗斯的核心                      |
| **留里克王朝**   | 👑 统治家族/王朝 | 862–1598年  | 北欧瓦良格人留里克建立，<br>→基辅罗斯<br>→莫斯科公国<br>→早期沙皇俄国 |
| **罗曼诺夫王朝** | 👑 统治家族/王朝 | 1613–1917年 | 留里克绝嗣后新王朝，<br>→沙皇俄国<br>→俄罗斯帝国                      |

> 💡 关键区分：**"罗斯/公国/帝国"是国号（政权实体），"留里克/罗曼诺夫"是王朝（统治家族）**

从**王朝**角度讲，十月革命前的俄国历史脉络比中国历史清晰多了：

```
留里克王朝（统治家族）
│
├─▶ 统治 基辅罗斯（882-1240）
│       └─ 蒙古入侵后分裂
│
├─▶ 统治 莫斯科公国（1283-1547）
│       └─ 1480年摆脱蒙古，1547年升级为"沙皇俄国"
│
└─▶ 统治 早期沙皇俄国（1547-1598）
        └─ 1598年绝嗣灭亡

[空位时期 1598-1613]

罗曼诺夫王朝（新统治家族）
│
├─▶ 统治 沙皇俄国（1613-1721）
│
└─▶ 统治 俄罗斯帝国（1721-1917）
        └─ 1917年二月革命终结
```

### 📅 极简版核心时间线

```
862年  留里克建立诺夫哥罗德公国 → 留里克王朝开始
  │
882年  迁都基辅 → 【基辅罗斯时期】开始（9-13世纪）
  │
1240年 蒙古西征，基辅罗斯灭亡
  │
1283年 蒙古扶植【莫斯科公国】（仍属留里克王朝统治）
  │
1480年 伊凡三世摆脱金帐汗国统治，莫斯科公国崛起
  │
1547年 伊凡四世加冕"沙皇" → 国号改为"沙皇俄国"
  │
1598年 留里克王朝绝嗣灭亡
  │
1613年 米哈伊尔·罗曼诺夫即位 → 【罗曼诺夫王朝】开始
  │
1721年 彼得一世称"皇帝" → 国号改为"俄罗斯帝国"
  │
1917年 二月革命，罗曼诺夫王朝终结
```

好的，现在带着这个梗概，我们可以直接进入俄国历史的海洋遨游了。后面会出现前述若干概念，但是会逐步增补。

## 罗曼诺夫王朝初期与国家奠基 (1613-1721)

罗曼诺夫王朝的建立标志着俄罗斯历史的一个新时代。尽管初期面临诸多挑战，但该王朝 通过持续的内部改革和对外战争，逐步强化了中央集权，扩大了国家疆域。彼得大帝的统 治是这一时期的巅峰，他发动了一场全面的、旨在模仿西方的“开明专制”改革，彻底重塑 了俄罗斯的国家面貌，将其推向现代化进程的快车道。

## 沙皇俄国的现代化与扩张 (1721-1801)

在彼得大帝奠定的基础上，叶卡捷琳娜二世的统治迎来了俄罗斯帝国的极盛时期。她通过**一系列成功的对外战争**，极大地扩展了帝国的版图和影响力。与此同时，帝国的官僚体系和贵族阶层也得到了进一步发展。然而，随着农奴制的固化和社会矛盾的积累，帝国也开始显露出衰落的迹象。

## 19世纪的改革、危机与革命前夜 (1801-1900) 

19世纪的俄罗斯帝国面临着深刻的内外挑战。一方面，帝国在拿破仑战争中遭受重创;另 一方面，国内的农奴制危机日益严重，社会变革的要求日益高涨。亚历山大一世转向保守 主义，而亚历山大二世则被迫进行了一系列激进的改革，尤其是1861年的农奴制改革。然而，这些改革并未能根除社会矛盾，反而催生了更为激进的革命力量，为帝国的最终崩溃 埋下了伏笔。

!!! note ""
    俄国历史自19世纪初始，便呈现为一连串革命浪潮和革命波谷的此起彼伏。这些浪潮一次比一次高。**第一次浪潮爆发于1825年，完全没有获得支持、彻底失败的12月党人哗变**。

    之后是尼古拉一世下的反动时期，**第二浪潮**在这一时期形成，缓慢渐进地发展。**在60年代，为自由主义改革所延缓**，同时得到加强，在民意党的活动中达到顶峰。 **1881年演变成对亚历山大二世的刺杀**。
    
    随之而来的低潮期已不如上一个时期那般持久。革命在九十年代聚集力量，并达到前所未有的高度，**第三次浪潮在1905年令人恐怖地剧烈爆发**。革命活动再次被镇压，后在战争期间再度兴起，最终在1917年获得成功（第二次革命）。 **第三浪潮首次获得民众的广泛支持，1905年革命被称为第一次革命**。

## 苏维埃俄国与苏联时期 (1917-1991)

1917年十月革命的爆发，标志着俄罗斯历史进入了全新的纪元。布尔什维克党推翻了临时 政府，建立了世界上第一个社会主义国家。在经历了内战、新经济政策和斯大林模式的确 立后，苏联成为一个超级大国。第二次世界大战期间，苏联付出了巨大牺牲赢得了胜利。 战后，苏联与美国形成了两极格局。赫鲁晓夫的“去斯大林化”和勃列日涅夫的停滞时期之 后，戈尔巴乔夫的改革最终导致了苏联的解体。
