# 云隐

标题是乱取的。

梳理一下 1966～1976 在华夏大地的风云际会与世事激荡。主要基于官方描述。我必须承认这是有偏的：但是历史就是这样一点一点地像裱糊匠一样裱糊出来的。我很喜欢那个笑话：<span style="color:red;font-weight:bold">历史就是死人名字</span>，这是很轻又很重的描写，启发于之，我试图总结记录一下<span style="color:red;font-weight:bold">人、事、物、职</span>，而这不可避免地会碰到<span style="color:red;font-weight:bold">时、空、生、死</span>，这或许就是我的本意。

本整理试图从不同的描述角度重整一下自己对这段历史的认知。错误难免。再次叠甲：个人整理，不代表任何观点。

## 按历史事件简单划分

### 1965-11-10：《评新编历史剧〈海瑞罢官〉》

北京。姚文元。《文汇报》，笔杆子的出山作，揭开序幕。

### 1966-02-12：《二月提纲》

北京。彭真（组长）、陆定一、康生、周扬、吴冷西。关于历史问题学术讨论的提纲。全名《文化革命五人小组关于当前学术讨论的汇报提纲》

> 《提纲》明确指出：​“要坚持实事求是，在真理面前人人平等的原则，要以理服人，不要像学阀一样武断和以势压人。​”

### 1966-05-07：《5.7指示》

因发布日期而得称 “五七”指示：教员对林彪报送的总后勤部《关于进一步搞好部队农副业生产的报告》作出的批示，提出全国各行各业都要办成“一个大学校”，既“学政治、学军事、学文化”，“又能从事农副业生产”，“又能办一些中小工厂”，“又能从事群众工作”，“又要随时参加批判资产阶级的文化革命斗争”。成为后来“五七干校”的开端。

### 1966-05-16：《5.16通知》

北京。政治局扩大会议。是 CR 发动的标志。推翻了《二月提纲》，同时伴随着对彭、罗、陆、杨的清算打倒。

[阅读全文](https://liyandi.gitbooks.io/maozedongxuanji/content/zhong-guo-gong-chan-dang-zhong-yang-wei-yuan-hui-tong-zhi-ff0d-4e94b7-yi-liu-tong-zhi.html)。

“阶级斗争扩大化的错误估计。”

### 1966-06-01：《横扫一切牛鬼蛇神》

北京。人民日报。[全文阅读](https://cn.govopendata.com/renminribao/1966/6/1/1/)

同日全国播发了聂元梓等人的大字报：《宋硕、陆平、彭珮云在文化革命中究竟干些什么？》（[全文阅读](https://cn.govopendata.com/renminribao/1966/6/2/1/#356770)，被陈伟“全国第一张马列主义的大字报”。

### 1966-08-05：《炮打死司令部：我的一张大字报》

教员。八届十一中全会。斗争的矛头直指中央。

[阅读全文](https://liyandi.gitbooks.io/maozedongxuanji/content/pao-da-si-ling-bu-ff0d-wo-de-yi-zhang-da-zi-bao.html).

### 1966-08-08：《十六条》

北京。八届十一中全会。

[阅读全文](https://liyandi.gitbooks.io/maozedongxuanji/content/zhong-guo-gong-chan-dang-zhong-yang-wei-yuan-hui-guan-yu-wu-chan-jie-ji-wen-hua-da-ge-ming-de-jue-ding.html)

!!! quote ""
    在当前，我们的目的是：
    1. 斗垮走资本主义道路的当权派，
    2. 批判资产阶级的反动学术“权威”​，
    3. 批判资产阶级和一切剥削阶级的意识形态，改革教育、改革文艺、改革一切不适应社会主义经济基础的上层建筑。
    4. 这次运动的重点，是整党内那些走资本主义道路的当权派，并要把那里的领导权夺回到无产阶级革命派手中。

### 1966-08-18：接见百万红卫兵小将

教员。北京天安门广场。自当日至 1966-11-26，共连续八次接见，总一千多万人次。

“孙悟空大闹天宫”的“天下大乱”。“不要如丧考妣，乱就乱他一阵子，我听说哪里乱了，我就高兴，这个乱是乱了敌人，乱了走资本主义道路的当权派。他们还在那里按部就班的工作，我就不高兴。”

语录、像章与军装。

### 1966-11-10：安亭事件

上海。王洪文（时任上海棉花十七厂保卫科干事）。成立“上海工人革命造反总司令部”。没有得到市委接见后，冲击上海火车站强行登车前往北京的列车。在列车被中央要求扣停后，于安亭卧轨拦车并攻击维稳干部，破坏全国交通运输。

“主要内容，就是要打倒市委，夺权！向市委书记曹荻秋夺权！”

张春桥背地指示并承认了“夺权”的合法性，承认了地方工厂、造反派的合理性。“巴黎公社起义”，“中国的十月革命”。

### 1967-01-12：“一月革命”

上海。张春桥、姚文元、王洪文。紧接着“安亭事件”后，又制造“康平路事件”，并最终在上海实现夺权，《文汇报》、《解放日报》。

### 1967-02-11：“二月逆流”

北京。京西宾馆、怀仁堂。一次在02-11，一次在02-16。党内的公开斗争。针对“踢开党委闹革命”、地方领导、军队机关干部受到冲击、身体暴力、“四大”、谁来领导军队里的革命、是否需要领导指挥等问题与“如何继续革命”等核心问题的激烈冲突。

军队元老：谭震林、陈毅、叶剑英、李先念、徐向前、聂荣臻、李富春。

中央文革：陈伯达、康生、江青、谢富治、张春桥。

“有些问题要好多年才能搞清。”

- 参考 1967-02-18《批评“二月逆流”》，[全文阅读](https://liyandi.gitbooks.io/maozedongxuanji/content/pi-ping-201c-er-yue-ni-liu-201d.html).

### 1967-03-19：“支左”，“三支两军”

北京。中央军委。《关于集中力量执行支左、支农、支工、军管、军训任务的决定》，同日，中央发出《关于停止全国大串连的通知》，指出：目前正在实现无产阶级革命派大联合，建立三结合的临时权力机构。继续停止全国大串连，取消原定的当年春暖后进行大串连的计划。

[阅读全文](https://zh.wikisource.org/wiki/关于集中力量执行支左、支农、支工、军管、军训任务的决定)。

全国280万解放军干部奔赴陌生的国家机关、工矿企业、农村、大中小学校中。

“理解的要执行，不理解的也要执行”！

!!! example ""
    由于教员和中央文革都没有明确指出“左派”的标准，许多部队开始都支持共产党员共青团员多的、​“出身好”的、劳动模范多的、拥护军队的一派。而这一派，往往因为“造反精神”不强，被对立面的一派称为“保守派”​。那些所谓“造反精神”强，敢冲、敢斗、敢闯的则被视为“革命派”​，即“左派”​。开初，许多军区都因支持了“保守派”​，犯了所谓“方向路线”错误而受到批判。
