# 我们时代的社会思考与阅读主张

## 思

!!! note "时代的社会变迁：1978-1998 年的转型轨迹"

    **1978-1998 年的社会转型塑造了我们时代的精神结构**——从封闭走向开放，市场化改革催生了中产阶级与新的社会规范。

    - **1978 年**：我们时代的起点，对知识的渴望前所未有，观念冲击开始
    - **1992 年**：自私自利和买卖变得正当（投机倒把罪废除），社会从封闭走向开放，成功学开始出现
    - **1994 年**：国税与地税改革，调动地方赚钱积极性
    - **1998 年**：城市住房体制改革，中产阶级开始出现，市民阶层形成

    文化镜像：《编辑部的故事》（1991 年）——中国大陆第一部情景喜剧，折射时代转型初期的社会心态。

!!! note "社会规范与知识积累"

    **社会规范在交流中产生，阅读构成知识积累**——带着社会思考阅读社科类内容。

    - 有人的地方就有江湖，有江湖的地方就有社会，社会就会有**社会规范**
    - 规范是在交流中产生的：手机外放从广泛接受到不被接受，体现了社会规范的演化
    - 为了减少社会矛盾，需要采取**社会规范**进行限制
    - 在社会层面看的书，一定要带着**社会思考**——受时代塑造，为社会思考

## 阅读的本质与碎片化问题

!!! question "什么是碎片阅读？"

    **碎片阅读的本质不是载体问题，而是思维姿态问题**——关键在于是否提供思考角度而非结果，是否需要筛选读者。

    - **载体迷思**：点开的东西就是碎片的吗？翻开的东西就不是碎片吗？
        - 点开之后如果只告诉你结果，那只是结果而非阅读
        - 弹出过程的书，不也是一种结果吗？
    - **深度阅读的特征**：
        - 看完 300 多本书从不从头看到尾——对"读完一本书"的定义要求不同
        - 把一本书读薄？还是把大脑读厚？（但很多时候分明脑子也读薄了）
        - 挑重点看，社科类文章看参考文献——在什么样的知识链条上看？
    - **文学艺术**主要是看一种"**体验**"

    **筛选读者的问题**：

    | 类型       | 特征                   | 是否需要筛选读者 |
    | ---------- | ---------------------- | ---------------- |
    | 媒体与学术 | 政治宣传、文学宣传     | 否               |
    | 市场营销   | 重复重复就是广告的特征 | 否               |
    | 社会思考   | 需要带着思考角度       | **是**           |

    - 阅读的矛盾：不可证伪的特性，需要一以贯之，但时代在改变
    - 愿意接受不同的观点，愿意看到一些不足之处
    - 思考的原命题一定需要放到不同的语境中来审视
    - **自由与平等的冲突**——经典的思想张力

## 如何读好一本书

!!! example "阅读的认识论：主动建构与逻辑自洽"

    **阅读是主动建构的过程**——需要逻辑自洽、多角度解构"正确"、从问题出发而非被动接受。

    ### 关于读书的认识

    - 别人读的别人讲的，从来都是别人的看法——你只能说别人讲的好，而**不知道书好不好**，一定要自己看
    - 人都会涉及有利于自己的方面，在向别人倾诉的时候，我们才会觉得他讲得有道理
    - 每个人在讲述的时候都在筛选信息，所以除非自己看，看到的都是被筛选的消息
    - 理论上，只要他讲的时候**逻辑自洽**，你就会觉得很有道理

    > **逻辑自洽而不是谈立场**

    ### 什么是好的文字？如何读好一本书？

    - **点开的东西，如果只提供给你结果，他不算是阅读**
    - 书需要给你提供一个**思考的角度**，而不是一个结果
    - 阅读构成了你的知识积累
    - 读者比作者聪明，但作者比读者考虑更加全面
        - 读者比作者有优越感
        - 看得越多，越会接近"**正确**"——但**正确是需要被解构的**
        - 至少会从别的角度看，会从多角度思考
    - 写出来必须在逻辑上自洽，必须改一些东西
        - 但很多东西就说不通，就是能写得通
    - **所有的了解都必须是自己去读的**，带着一本书，从一个角度去看——到底能不能解决问题？
