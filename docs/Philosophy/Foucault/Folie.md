# 疯狂

**应该撰写这另一种疯狂的历史——由于这种疯狂，人们在监禁他们邻人的至高理性的活动中，通过非疯狂的无情的语言相互交流、相互确认** ；应该重新经历这种阴谋在真理王国中被最终确立以及被反抗的激情唤醒之前的时刻。在历史中努力回到疯狂历史的零度，在那儿，它是未被区分的经验，是尚未分割的分割自身的经验。

在精神病的沉寂世界中，现代人不再与疯子交流：==一方面是理性的人，他们给疯子派去医生，只认可一种以疾病的抽象普遍性为中介的关系== :另一方面是疯狂的人，**他们只能借助于同样抽象的理性与另一部分人交流，这种理性就是秩序、肉体和道德的约束、匿名的集体压迫、必须服从的要求**。共同的语言是没有的，或者毋宁说不再有了;18世纪末疯狂作为精神病的确立，标志着对话的中断，宜告了分隔已经确立，疯狂与理性曾经用来交流的不完善的、缺乏固定句法的、略显结巴的语汇被全部归于忘却，精神病学的语言——是理智关于疯狂的独白——只能在这样的沉寂之上建立起来。

欧洲人自中世纪以来就与某种被混乱地称作疯狂、精神错乱、丧失理智的东西发生联系，也许像与苏格拉底对的智者将某些东西归于病态的威胁一样，<span style="color:red;font-weight:bold">西方理性正是要将自己深度中的某些东西归于这种模糊的存在</span>。无论如何，对于西方文化来说，<span style="color:red;font-weight:bold">理性与非理性的关系构成了它的独特性的一个维度</span>。

**要撰写一部限度史**——关于那些模糊的、一经发生必然就被忘却的事件的历史，由于这些事件，一种文化推斥某些东西，使它们成为它的外部；在文化的整个历史中，被挖掘出来的空洞、孤立它的空白空间确定了它和它的价值。因为它的价值，它接受并在历史的连续性中坚持它们；但在我们要谈及的领域中，文化作出基本选择.实施分割，这种分割为它提供了实证性外貌；这里存在着文化从中得以形成的原初深度。

在西方世界的经验限度中心所突显的，**当然是悲剧本身的经验限度——尼采曾揭示，西方世界的历史由之发生的悲剧结构就是对悲剧的拒绝、忘却和默默的沉落**。其他经验环绕着悲剧的结构，是中心，因为在历史对悲剧的拒斥中，悲剧的结构已将悲剧性编入了历史的辩证法。

**在西方理性的普遍性中，有这样一个分割，那就是东方**：东方，被当作起源，被想象成骇人的一点，由此衍生出乡愁和回忆的允诺，东方，壆现在西方殖民理性面前，却又永远无法接近，因内它总是限度：是西方形成和标划分线时的最初的黑夜。

同样应描述其他分割: ... ==对欲望的幸福世界的悲剧性分割，正是西方世界的限度和它的道德起源。最后，也是首要的，应谈论疯狂的经验==。

---

疯狂，在它最普遍的形式中，在它最具体的形式中究竟是什么，<span style="color:red;font-weight:bold">为什么知识从一开始就放弃了对它的全部控制？毫无疑问，除了著作的缺失，它什么也不是。</span>

!!! quote ""
    历史只是在一种历史缺失的基础上，在巨大的喃喃低语的空同中才可能，沉默把这缺失当作使命和真理来守候:“我把你建造的城堡称作荒芜，把这声音称作静夜，把你的面容称作映失。“这个模糊的领域意义含混:是纯粹的起源，因为正是由此出发一点点克服句法和词汇的混淆，才产生出历史的语言。

    > 反映一种历史的空洞、文本的缺失和模糊含混意义的大行其道，因此有必要为其划定限度，研究这些没有被很好揭露的疯狂。

尽管理性知识把疯狂当作疾病来消除、治愈，但单是这个结构就表明了一个普遍事实：==在我们的文化中，没有疯狂就没有理智。贯穿西方历史的疯狂的必然性是与这个决定互伴相联的== ... 简言之，==疯狂的必然性与历史的可能性相联系==。

==撰写疯狂史，即是要对历史集合体——观念，机构、法律和警察制度、科学概念——进行结构性研究==，这种集合体捕捉住疯狂，但也使疯狂自身的原生状态再难恢复;由于这种不可接近的原初纯粹性的缺乏，**结构性研究应回溯到那同时使理智与疯狂相联系和分离的决定；应尽可能揭示永恒的交流、共同的模糊根源、那给予意义和荒谬之间的同一与对立以意义的原初对峙**。


!!! example ""
    从威依斯(Wilis)到毕奈尔(Pinel)，从俄瑞斯忒斯(Oreste)的迷狂到聋人之屋、到朱丽叶(Juletle)一恰恰掩盖了这个阶段，其闻疯狂与理性的交流彻底改换了语，在疯狂史中，==有两件事清楚地显现了这种变动：1657年总医院的设立和对穷人的“大监禁”；1794年毕塞特囚徒的解放==。在这两个独特的、对称的事件之间，发生了某些事情，其**模糊性**令医学史家感到窘迫：是专制政体中一会儿依据这，一会几依据那的任意压制，还是科学和仁慈在疯狂的实证或理性方面取得的进步、发现?

应该要有一种足够中立的（独立于科学术语、独立于社会或道德选择的）语言，以便**尽可能地接近那些原初混乱的词**，以便取消现代人在自信与疯狂之间所保持的距离；应该要有一种足够开放的语言，以便**忠实地记录那些决定性的、为我们造就了疯狂与理智的真理的言谈**。
