# 性存在（Sexuality）

---

!!! info "书籍信息与核心背景"

    **本书是牛津通识读本系列作品，由韦罗妮克·莫捷撰写，系统梳理了性存在概念的历史演变与当代政治意涵。**

    现代西方社会在性领域发生了三大革命性转变：

    1.  **性的世俗化**：性与宗教价值观彻底分离，脱离基督教性文化以生殖为唯一合法功能的窠臼
    2.  **性的自由化**：社会对计划生育、人工流产、离婚、婚前性行为、同居和同性恋的接受程度大幅提高
    3.  **家庭模式危机**：人际关系模式的深刻变革动摇了传统家庭的社会基础

!!! abstract "前现代性观念的历史演变"

    **前现代社会的性观念经历了从古希腊阳具中心主义到古罗马伦理再到基督教性道德的演变过程，其核心始终与社会权力结构紧密绑定。**

    古希腊的性理解是彻底的阳具中心主义的，性的唯一定义是阴茎的侵入，亲吻、爱抚等身体接触仅被视为示爱表现而非性行为。性秩序与政治社会秩序直接混同，侵入行为本身象征着男性身份与社会地位，与被侵入对象的性别无关。

    - 古希腊人不依据生理性别身份认同自我，而是将社会性别身份置于核心地位，性文化反映了男性公民的社会权力，以男性享乐为核心
    - 古希腊的"希腊式恋爱"指成年男性对少年男子的情爱，被视为自然高尚的行为，因为爱与自己平等的生物比爱低等生物更为高尚
    - 医学层面认为女性阴冷潮湿的身体需要男性热量补充，子宫处于漂移不定状态，需要男性精液维持稳定，否则会四处移动导致女性窒息
    - 法律层面，强奸不被视为对女性本人的犯罪，而是对其丈夫、父亲或男性监护人的财产侵犯，通奸仅定义为涉及已婚女性的性行为，是最受谴责的性犯罪
    - 社会阶层层面形成明确的性角色分工："高级妓女提供享受，侍妾满足身体日常所需，妻子负责诞育合法后代并保卫家庭"

    古罗马时期基本延续了古希腊的性伦理，但随着政治权力向教会转移，基督教开始对性欲进行严格监管：

    - 独身和贞洁被视为最高行为规范，性欲因使人耽于世俗俗务而备受指责，基督教对性的敌意源于将人们从世俗束缚中解放出来的宗教任务
    - "原罪"教义将亚当夏娃被逐出伊甸园的原因归结为性，早期神学家甚至认为如果可以选择，人类应该通过捏泥巴而非性交繁衍后代
    - 教会最终妥协允许婚姻内的生殖性性行为，但将所有非生殖性性行为视为罪恶

!!! note "现代"性存在"概念的诞生"

    **"性存在"（Sexuality）作为一个独立概念诞生于19世纪，标志着性从宗教道德范畴进入科学医学范畴，同时也开启了对非规范性行为的医疗化和病理化过程。**

    根据《牛津英语词典》，"性存在"一词于1879年正式进入英语语言，其当代意义为"对性能量的拥有状态，或对性进行感受的能力"。这一概念的形成与当时的性科学研究紧密相关：

    - 精神病学开始对各种"异常"性行为进行分类和诊断，同性恋被从一种行为模式重新定义为一种天生的病理特征
    - 自慰被污名化为"俄南之罪"，被视为导致各种身体和精神疾病的原因
    - 弗洛伊德的童年性欲理论进一步强化了性在人类心理发展中的核心地位
    - 女性的性需求被系统性病理化，整个19世纪"女性歇斯底里症"诊断数量大幅增长，被定义为女性性欲过盛得不到满足导致的精神错乱，治疗方式是医生对生殖器进行人工按摩直到达到"歇斯底里痉挛"（即性高潮）

!!! critique "女权主义对性存在的政治批判"

    **20世纪女权主义运动对父权制性规范进行了系统性批判，提出了性解放、身体自主等核心主张，同时也产生了包括政治女同性恋主义在内的激进策略。**

    女权主义性批判的发展脉络：

    - 19世纪末的社会运动推动英国1885年刑法修正案出台，将嫖娼列为非法，并将女性最低合法性行为年龄从13岁提高到16岁
    - 自由恋爱运动反对建立在男性对女性经济和身体掠夺基础上的婚姻契约，主张将自然性欲从资产阶级压迫中解放出来
    - 性学家雪儿·海蒂的研究发现大多数女性完全拥有性高潮能力，但这种高潮通常无法通过常规性交实现，挑战了传统的性满足叙事
    - 激进女权主义提出"政治女同性恋主义"概念，将其视为反抗男权的关键政治手段，认为男性是敌人，异性恋女性是敌人的帮凶，甚至提出"倘若女权主义者还在与男性同床，便是在将自己最活跃的能量输送给压迫者"
    - 后殖民女权主义对主流女权主义提出批判，指出部分西方女权主义者运用美国种族中心主义视角审视非洲文化行为，带有文化帝国主义和新殖民主义色彩

!!! success "当代性政治与未来走向"

    **20世纪中后期的性政治运动在挑战传统性规范的同时，也面临国家权力的干预和意识形态的争议，酷儿理论的出现开启了性政治的新范式。**

    现代性政治的核心矛盾与发展方向：

    - 国家权力对性的干预不断增强，20世纪的优生学运动将种族延续作为最高目标，甚至部分同性恋优生学家认为同性恋者携带"劣等"基因，更有可能生下智力低下的孩子，因此反对同性婚姻
    - 20世纪七八十年代的女同性恋分离主义运动不仅寻求组织独立，甚至寻求地理上的独立，将性别对立推向极端
    - 酷儿理论与酷儿政治的出现标志着性政治的新转向：不再要求在私人领域享受孤立的性自由，而是通过一系列行动要求公共领域的去异性恋化，强调对主导性社会意义和身份的"永久反抗"与颠覆
