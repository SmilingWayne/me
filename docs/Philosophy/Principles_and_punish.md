# 规训与惩罚

这本书，最晚在我本科读大三的时候就翻过，毫无疑问当时完全看不懂。那个时候又在限制自己不乱买书。于是一直没有买纸质版，留着电子版翻了好久。现在（研二）了，沦落为一个悲催的北漂，偏远狭小的居室里没有纸质书，于是毅然决然入手了一本，与之对应还买了一本《浮士德》。两本书买回来了也没立刻看，拖沓了好久，五一去天津，顺手带上了它。坐在海河边吹风的时候，想到自己丰满的行囊里还有这样一本书，于是找了家书店（不推荐，商业气太厚）坐了一下午，从头到尾读完了这本书：**我决定以后每次出门旅行都要带一本书**。

我一直不愿意对哲学书籍“大放厥词”，但是这本书过于值得说道，以至于我斗胆可以跑来咕哝咕哝。

从最表面的意义上，福柯认识中的社会充满了冷酷与恶意，将支配和被支配关系渗透到每一个毛细血管里，他愿意用微观政治学来刻画社会对个体的掌控，仿佛政治、权力在这一刻活了过来，无处捉摸却又无处不在。他不吝啬于宣扬“世界就是个大军营！大监狱！管教所！” 甚至连知识本身也是权力在生产的结果。言语间兴奋得仿佛他刚刚拧下绞刑架的螺丝，把全景监狱的最后一个电路接上，让监控摄像头监控着每一个人，行为和时间被切分，人被Normalized成一个个奇奇怪怪的生物。

一个被忽视的事实是，同样的观点放在别人嘴里说出来会显得狂妄自大，但是福柯成功给现代人洗了脑。重要原因在于他摸到了现代社会中每个原子化个体的G点：每个在物质充足的原子人都感受到了某种来自系统的掌控感。大多时候无力感只是因为个体能力有限，但是系统的作用就在于拥有时刻“能够让人害怕”的能力。剥掉外衣，人们看到了权力的耀武扬威。

权力不会耀武扬威的。权力通过逐渐膨胀的能力不断延伸自己的触手，伸到方方面面。监控摄像头？福柯说太Naive了。起码你得看到监控摄像头。真正的控制是在大脑皮层的控制。

事实上，个体已经被规训了。东亚人喜欢“辩证”，这也是一种规训：“发挥规模效应”，这样大家步伐一致，脑筋里的神经元都一致地移动。作为驳论方，你可以从反方向觉得福柯在安那其，瞎搞，就这还哲学家？我深以为然。但你必须承认这是一种很强大的安那其。事实上那些有颠覆性的东西一开始都以一种安那其的方式出现，但是通过**再生产**，原本对安那其变成一种有进步意义的玩意。

我应该在这里住嘴了。我只是震撼于本书对原子化个体在现代社会对处境进行的解释。

## 酷刑


!!! quote "==观点一== ：公开处决、对肉体惩罚的酷刑、断头台，其本身就是就是一种仪式、一种公共景观、一种政治运作，旨在彰显权力（王权、司法权）。但是19世纪开始，这种景观消失了，反而被非肉体惩罚体系包裹起来。这就是以所谓“人道”取代“酷刑”的过程。"
    曾经降临在肉体上的死亡，被代之以深入灵魂、思想、意志的惩罚。

    但是，这种公开处决有时候会有副作用：民众被召集来观看旨在恫吓他们的场面，而他们则可能表现出对惩罚权力的拒斥，导致社会骚动。这是因为民众的角色是多义的



!!! quote "==观点二== ：权力和知识是相互连带的。不是知识主体的活动产生某种有助于权力或者反抗权力的知识体系，相反，权力-知识，贯穿权力-知识和构成权力-知识的发展变化和矛盾斗争，决定了知识的形式和可能的领域。"
    简单说来，知识不是平白无故被人总结出来，而是它总是被权力制造、鼓励的。即 **“权力对知识的再生产”**。这是福柯十分经典和著名的论点。

    > 我们不应该把对知识的分析建立在“认识主体相对权力体系是否自由”这一问题上。
    > 
    > 我们应该抛弃那些想法，即“弃绝权力乃是获得知识的条件”。

## 惩罚 

### 普遍的惩罚


!!! info "对第一章“人道”取代公开处决的再讨论：现代惩罚制度与规范体系构建"

    “其首要的目标是：**使对非法活动的惩罚和镇压变成一种有规则的功能**，与社会同步发展，**不是要惩罚得更少些，而是要惩罚得更有效率些**，或许应减轻惩罚的严酷性，但目的在于使惩罚更具有普遍性和必要性，==使惩罚的权力更深地嵌入社会本身。”== 。

    这句话的重点在于，高效的惩罚，而高效的惩罚首先就需要**规范化的功能**。

    --- 

    在旧体制下，犯人的肉体变成国王的财产，君主在上面留下自己的印记和自己权力的效果，现在，它将是社会的财产，集体占用的对象。

    **人们应该设想一个惩罚之城。**

!!! example "时代背景：资产阶级权力对犯罪和非法活动的重新定义与改造，对财产相关犯罪的再造。"
    在旧制度下，**各个社会阶层都有各自被容忍的非法活动的余地，每个阶层也都有自己独特的必要的非法活动。** “在法律和习俗的边缘处，一块被宽容的空间”。这种非法结构活动也随着资本主义社会的发展而被改造。

    > 一方面下层阶级最能接受的是对财产的非法占有：所有权的剧烈转移；另一方面资产阶级则要为自己保留权利的非法行使，也就是规避自己的规章和法律，巧妙地利用法律空隙使自己获得巨大的经济活动地盘。
    >


!!! note "观点：精心计算的惩罚体系，以及通过这种体系束缚奴隶们的思想"
    我们可以发现，在这种刑罚人道化的背后，所隐含的是所有那些认可、或更准确地说是要求“仁慈“的原则，**是一种精心计算的惩罚权力经济学**，但是这些原则也引起了权力作用点的变化：不再是通过公开处决中制造过度痛苦和公开羞辱的仪式游戏运用于肉体，而是运用于精神，更确切地说，==运用于在一切人脑海中谨慎地但是也是必然地和明显地传播着的表象和符号的游戏==。“权力技术的相关物”：旧的惩罚“解剖学”被抛弃了，但是我们真的进入了非肉体惩罚的时代吗？

    愚蠢的暴君用铁链束缚他的奴隶，而真正的政治家则用奴隶自己的思想锁链更有力地束缚他们。正是在这种稳固的理智基点上，他紧紧把握着锁链的顶端。这种联系是稳固的，因为我们不知道它们是用什么做成的，而且我们相信它是我们自愿的结果。==绝望和时间能够销蚀钢铁的镣铐，但却无力破坏思想的习惯集合==，而只能使之变得更紧密，**最牢固的帝国的不可动摇的基础就建立在大脑的软纤维组织上。**

### 总结

总之，可以说，在18世纪晚期，人们面对着三种组织惩罚权力的方式。**第一种是依然发挥作用的，基于旧的君主制度的方式。另两种都基于一种认为惩罚权利应属于整个社会，应具有预防和改造功能的功利主义观念**。但这二者在所设想的机制方面迥然不同。

广义地看，人们可以说，在君主制度中，惩罚是君权的一种仪式。它使用报复的仪式标志，对犯人的肉体施加报复。它是君主及其权力的物质表现。它是不连贯、不规范的，总是凌驾于自身的法律之上，它在众目睽睽之下制造强烈的恐怖效果。而（第二种方式）主张改革的法学家则<u>把惩罚视为使人重新获得权利主体资格的程序</u>。惩罚不应使用标志，而应使用符号，即一系列被编码的表象。这些表象应能得到快速的传播，并能最普遍地被目睹了惩罚场面的公民所接受。但是（第三种方式，即监狱制度）惩罚被视为对人体进行强制的战术。

它通过在习惯、行为中留下的准则，施展**训练肉体**（不是符号）的方法。它以建立一种特殊的管理刑罚的权力为先次条件。

|       |       第一种 |                       第二种 |             第三种 |
| :---: | -----------: | ---------------------------: | -----------------: |
|   -   | 君主及其威力 |                   社会共同体 |           管理机构 |
|   -   |         标志 |                         符号 |               痕迹 |
|   -   |         仪式 |                         表象 |               操作 |
| 对象  | 被消灭的敌人 | 处于恢复资格过程中的权利主体 | 受到直接强制的个人 |
|   -   | 受折磨的肉体 |       具有被操纵的表象的灵魂 |       被训练的肉体 |

**它们是惩罚权力运作的三种方式，是三种权力技术学**。这样，就出现了下列问题：为什么第三种方式最终被采纳了？惩罚权力的强制的、肉体性的、隔离的、隐秘的模式，为什么会取代表综的、戏剧性的、能指的、公开的、集体的模式？为什么休罚（不是酷刑）以监狱为制度依托，取代了惩罚符号的社会游戏和冗长的传播符号的节日？

### 训练的手段

强调的是“纪律”的构建方式。

#### 层级监视

“监视站”。分层的、持续的、切实的监督，可以使与之相关的权力机制变得重要。举例涉及：军营、学校的建筑设计。

#### 规范化裁决

属于纪律的一种特殊的惩罚方式，独立于法律体系，独处于小团体中的规则、规矩。能够缩小所约束群体之间的差异，能够反映同一个群组的差异，分门别类进行评级，进行奖励和惩罚。

> 总之，在规训权力的体系中，惩罚艺术的目的既不是将功补过，也不仅仅为了压制，而是把个人行动纳入一个整体。他**根据一个通用的规则来区分个人**，该准则应该是一个最低限度，一个人必须努力达到的适当标准。
> 


#### 检查 （Examination）

把层级监视的技术恶化规范化裁决的技术结合起来，是**一种追求规范化的目光**，一种能够导致定性、分类和惩罚的监视。它**确立了个人的能见度，因此人们可以区分和判断个人。**

> 检查处于使个人成为权力的后果与对象、知识的后果与对象的程序的中心位置，与层级监视和规范化裁决结合起来，就确保了重大的规训功能：分配和分类。**最大限度地榨取和利用时间。**
>
> 通过探求最佳的能力组合，以及随之而来的具有单元性、有机性和创生性的个性制作。这样纪律也被仪式化了：纪律是一种针对个人差异的权力运作方式。

### 全景敞视主义

!!! note "瘟疫来袭时的措施：严格的空间管理、不断的登记体制、不断的监督。"

!!! quote ""
    规训”既不会等同于一种体制也不会等同于一种机构。它是一种权力类型，一种行使权力的轨道。它包括一系列手段、技术、程序、应用层次、目标。它是一种权力“物理学”或权力“解剖学”，一种技术学。它可以被各种机构或体制接过来使用，

这种全景敞式监狱的主要后果：在被囚禁者身上造成一种有意识的和持续的可见状态，从而确保权力自动地发挥作用。

这是一种**重要的机制**：因为它使权力自动化和非个性化，<u>权力不再体现在一个人身上，而是体现在对肉体、表面、光线、目光的某种统一分配上，体现在一种安排上</u>。

**一种虚构的关系自动地产生出一种真实的征服。** 通过一种外在的、隐含的、精心构造的约束，实现对人高效的、轻便的、彻底的控制。

### “监狱”


我的论述就以这篇未署名的通信作为结束。我们此时已远离刑轮、绞刑架、示众柱星罗棋布的酷刑国度，我们也远离大约五十年之前改革者的梦想——惩罚之城 ... **摘自《法朗吉》的上述文字使我们想到某些更重要的东西：位于这座城市中心的，而且似乎是为了恰当地控制这个中心的，不是“权力中心”，不是一种武力网络，而是一个由不同因素组成的复杂网络：高墙、空间、机构、规章、话语**。

（监狱）占据着中心位置，但它不是茕茕子立，而是与一系列的“监狱”机制相联系。这些机制都是用于减轻痛苦、治疗创伤和给予慰藕的，因此表面上与监狱迥然有异，但它们同监狱一样，<u>都往往行使着一种致力于规范化的权力。</u>这些机制不是被用于对付对“中心”法律的冒犯，而是被用于生产机构:“商业”和“工业”，用于对付一系列复杂的非法活动。这些非法活动具有各式各样的性质与根源，有特殊的谋利作用，**惩罚机制对付它们的方法也是各式各样的**。追根究底，统辖着所有这些机制的不是某种机构的统一运作，而是进行战斗的必要性与战略准则。因此，把这些机构说成是压制、排斥、制造边缘状态的机构的种种观念，不足以描述出处于“监狱之城”核心的居心叵测的怜悯、不可公开的残酷伎俩、鸡零狗碎的小花招、精心计算的方法以及技术与“科学”等等的形成，所有这一切都是为了制造出受规训的个人，这种处于中心位置的并被统一起来的人性是复杂的权力关系的效果和工具，<span style="color:red">是受制于多种“监禁”机制的肉体和力量，是本身就包含着这种战略的诸种因素的话语的对象。在这种人性中，我们应该能听到隐约传来的战斗厮杀声。</span>

> 每一个监狱，又都是一个更大监狱的牢房。
