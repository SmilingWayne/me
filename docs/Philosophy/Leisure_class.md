## 摘抄

### Chapter. 1 导言

有闲阶级制度最充分的发展形态出现在未开化（Barbarian）文化的较高级形态：

1. ==上层阶级按照习俗得以免于或者不得从事生产性职业，而从事带有某种程度的荣誉感的职业==；
   1. 上层阶级女性通常得以免于从事生产性职业，男性不仅得以免于生产性职业，而且**按照惯例习惯不得从事任何生产性职业**；
2. ==体力劳动、生产作业以及任何与谋生相关的日常工作直接相关的职业，都专属于下层职业==；

如果在文化尺度上更进一步，来到“蒙昧”群体（savage）中，职业的区分反而更加不明显：

这些群落的人数不多，结构简单古老，通常社会和平，成员定居，群体贫穷：**私有制不是他们经济制度的主要特征**。

!!! note "有闲阶级制度稳定的必要条件显然有："
    1. <span style="color:red;font-weight:bold">群落必须有掠夺的生活习惯</span>；
    2. 谋生必须足够容易，令群落当中相当一部分成员得以免于持续性日常劳作；

    ---

    原始社会群体的活动区分于两种类型：==掠夺和生产==。

    生产是制造一种新东西的努力；掠夺是“其他能动者用于其他目标的能量，被转化为自己目标的能量”。


在这样的掠夺性狩猎群体中，强壮男性的职务是战斗和狩猎，而女性则被分配到“其他需要做的工作”，此时男性的战斗和狩猎具有共性，**都是掠夺**。任何不涉及展现实力的工作，对男性而言都变成毫无价值的。

> 区分和平文化和掠夺文化：根本区别是精神上的区别，而不在于物质上的区别：精神状态的改变是物质生活改变的结果：<u>只有当生产方式高度发展，在供给谋生者存活之外，尚有足够的余量值得争斗之时，掠夺才有可能成为任何群体或者任何阶级的习惯性和常规性资源</u>。

### Chapter. 2 金钱竞赛

对女性的所有权始于低级未开化文化阶段，这是以捕获女性俘虏为开端，作为战利品，又进一步发展，扩大为对女性以外的俘虏和弱势人群的奴役，以及对其他女性（战利品）的所有制婚姻 ... **其原因都是因为占据主导成功优势的男性想要展示其战利品的结果。**

<span style="color:red;font-weight:bold">所有制起源的动机是竞赛。</span>

通过先祖或者其他先人财产的被动转移而获取的财富，甚至变得比拥有者通过自身努力而获取的财富**更具有荣誉性**。

（这种竞赛造成）实力和掠夺可能仍是获得公众崇高敬意的基础，但是**拥有财富则成为普通程度的名声以及无可指责的社会地位的基础**。

一旦拥有财产成为公共敬意的基础，也就成了自尊心的自得心态的必要条件。为了获得心里平衡，每个人都必须获得与处于同一阶层的其他人拥有的同样多的物品，而如果拥有的物品多于其他人时，将产生极大的满足感。

**... 每个人都将与自己与旁人进行比较，形成新的阶层分级 ... 这种不公性比较是永无止境的 ... 进行这一比较的的人永远不会得到满足 ... 他在金钱声誉的斗争中永远希望达到比对手更高的地位。**

没有一种方法能够带来完全的满足。

在很大程度上，==金钱竞赛决定了人们在个人享受和体面生活方面的消费方式和消费品选择==。

另一个程度上，财富带来的<span style="color:red;font-weight:bold">权力</span>也是积累的动机：追求成就和厌恶徒劳无功的倾向仍然是经济动机的基础。

### Chapter. 3 炫耀性有闲

<span style="color:red;font-weight:bold">为了获得并保持声誉，仅仅占有财富和权力是不够用的。财富和权力必须提供证据，只有通过这些证据才能享受声誉。</span>

==普通人都会因为体面的环境和免于从事低贱的工作而感到身心舒畅==。自尊心也会获得提升：如果让他被迫放弃他所习惯的体面生活的标准，不管是在生活用品方面还是在日常活动的种类和数量方面，他都会感觉其作为人的尊严受到了轻忽。

这种自古以来关于生活方式的尊卑的理论区别，今天还在发挥作用：几乎所有的**上层阶级都会对粗俗的劳动形式抱有本能的厌恶之情**。我们清晰地意识到他们有一种形式上的不洁感：<u>一些通常仆人履行的职责不可避免地带有精神污染。</u>

==一定程度的有闲以及不接触直接服务于日常生活的生产进程，总是被有识之士视为受尊敬的、完美的、乃至无懈可击的人类生活的前提==：

<u>炫耀性的不劳动成为经济实力和崇高声誉的常规性指标，反过来，劳动成为贫穷的证据，不可避免地成为不光彩之事，哪怕劳动在古早文化阶段不被视为有失体统，结果依然如此</u>。

“炫耀性地避免所有实用性职业，成为有闲阶级生活的代表性特征”。

有闲阶级从事诸如祭祀、政务、体育、宗教，表面目的肯定不是通过生产性工作增加财富。免于劳动不仅是光荣或者值得赞许的行为，还逐渐成为体面的必要条件。

只要炫耀性有闲的规则有机会展示得淋漓尽致，就会出现一个次生的、在某种意义上的虚假的有闲阶级：穷酸落魄、朝不保夕，但是精神上却不愿意忍辱从事牟利事物。
